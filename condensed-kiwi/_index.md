# Condensed Kiwi

그래픽 디자인 팀. 2025년 12월 결성.

## 멤버
- 혁 (@hyuk.xyz)
- 하진
- 미주

## 현재 상태
- 작업 방향 논의 중
- 퍼블리싱된 작업물 아직 없음

## 운영 원칙

### 1. 프로젝트별 디렉터 제도
- 각 프로젝트마다 **책임자(디렉터)**를 지정한다
- 디렉터가 방향, 일정, 최종 결정권을 갖는다 — 합의제가 아니라 **위임제**
- 나머지 멤버는 협력자로 참여
- 디렉터가 피드백 요청 후 48시간 내 응답이 없으면 단독 결정 가능

### 2. 혁의 군복무 기간 제약 반영
- 혁이 디렉터인 프로젝트 → 기획/컨셉/디렉션 중심, 실행은 하진·미주에게 위임
- 하진 or 미주가 디렉터인 프로젝트 → 혁은 리뷰어 역할, 비동기 피드백
- 아이폰 작업 가능 범위를 고려하여 역할 배분

### 3. 프로젝트 규칙
- 산출물 단위를 작게: 2주 이내 완료 가능한 것부터 시작
- 결과물이 명확해야 함 (포스터 1장, 캐러셀 1세트, zine 1권 등)
- 완벽보다 완료 — 첫 아웃풋을 내는 것이 최우선

### 4. 커뮤니케이션 루틴
- 주 1회 체크인: "이번 주에 한 것 / 다음 주에 할 것" 한 줄씩
- 2주 연속 진행 없으면 프로젝트 보류 선언
- 보류된 프로젝트는 재개 의사 확인 후 계속 or 종료
