# 비평 — CK 브랜딩

## 핵심 주장
CK는 NotoSans 포크로 자체 폰트를 만들고, 다국어 피드를 통해 폰트를 노출시키며, 민희진식 단계적 통제 전략으로 폰트를 배포해 시각적 정체성을 확산한다.

## 범위 경계

**다루는 것**:
- CK 인스타그램의 언어 전략 (영어 제1언어)
- 비주얼 아이덴티티 방향 (Negative/Positive 확정)
- 팀 → 개인 수혜 구조 설계
- 벤치마킹 대상 분석
- **CK 폰트 전략 (NotoSans 포크 → 다국어 피드 → 배포)**
- **민희진/오케이레코즈 벤치마킹**

**다루지 않는 것**:
- 개인 계정의 운영 전략 (각자의 영역)
- 상세 마케팅/그로스 전략 (CK는 에이전시가 아님)
- 수익화 모델 (아직 단계가 아님)
- **타입 디자인 실무** (글리프 드로잉, 커닝 등 — 실행 단계의 영역)

## 가장 약한 고리

1. **NotoSans 파라미터 조정만으로 정체성 차별화가 가능한가?** 이것이 현재 리서치의 가장 취약한 지점. wght/wdth 축 조정은 시각적으로 NotoSans와 구별 불가능할 수 있다. "이건 NotoSans잖아"라는 반응이 나오면 전략 전체가 흔들린다. 글리프 레벨 수정까지 갈 건지, 아니면 맥락적 차별화(= 폰트 자체보다 사용 방식으로 정체성을 만든다)로 갈 건지 결정이 필요하다.

2. **1차 상품의 부재.** 민희진 벤치마킹을 하면서도, 민희진 모델의 핵심 전제(1차 상품이 존재해야 비주얼 시스템이 작동)에 대한 답이 아직 없다. 다국어 피드만으로 충분한가, 아니면 CK의 디자인 작업물이 선행되어야 하는가?

3. **"팀 → 개인 수혜 구조"의 전제 조건** (기존 약점 유지). CK 계정 자체가 먼저 콘텐츠 볼륨과 퀄리티를 갖춰야 함.

4. **OFL 라이선스와 배포 전략의 충돌.** NotoSans 포크는 OFL 조건상 폰트 단독 판매 불가. 유료 배포를 고려한다면 NotoSans에서 충분히 벗어나야 하는데, 그러면 NotoSans의 다국어 커버리지(최대 장점)를 유지할 수 있는가?

## 반론

**"민희진 모델이 CK에 적용 가능하다"가 틀렸다면:**
- 민희진 모델은 **자본과 인력**이 전제다. 오케이레코즈는 법인 자본금 3,000만 원 + 가로수길 사옥 + 전문 인력. CK는 3인 팀 + 군복무 제약 + 아이폰. 스케일이 다르다. 민희진의 "모든 접점 통제"는 CK 규모에서 실현 가능한가?
- 민희진의 정반합은 **기존 산업의 관습이 뚜렷할 때** 작동한다 (K-POP의 과도한 세계관 → 뉴진스의 자연스러움). CK가 속한 "소규모 디자인 콜렉티브" 씬에서 뚜렷한 '정(正)'이 있는가? 반대할 대상이 없으면 정반합도 없다.

**"NotoSans 포크가 전략적이다"가 틀렸다면:**
- NotoSans의 디자인 철학은 **중립성(neutrality)**이다. 중립적인 것을 약간 비튼 결과물도 여전히 중립적일 수 있다. 차라리 처음부터 강한 성격을 가진 폰트(Druk, Dharma Gothic 등)를 포크하거나, 완전히 새로 만드는 것이 정체성 측면에서 유리하지 않은가?
- 다만 반론의 반론: NotoSans를 선택한 것은 다국어 커버리지 때문이지 디자인 성격 때문이 아니다. 150개 문자 체계를 지원하는 폰트는 NotoSans 외에 사실상 없다.

**"다국어 피드가 그로스 전략으로 유효하다"가 틀렸다면:**
- 알고리즘이 언어별로 콘텐츠를 서빙하는 것은 사실이나, 각 언어 포스트가 해당 언어권에서 "발견"되려면 해시태그, 인게이지먼트 등 다른 신호도 필요하다. 30개 언어로 올려도 각 포스트의 도달이 극히 낮을 수 있음.
- 30개 언어 포스트가 피드 그리드를 점령하면, 이후 실제 작업물을 올릴 공간이 부족해진다. 론칭 캠페인으로서의 시점과 분량 조절이 필요.

## 복잡도 경고

**주의 필요.** 리서치가 세 방향으로 동시에 확장되고 있다:
1. 인스타그램 퍼블리싱 전략 (언어, 빈도, 포맷)
2. 비주얼 아이덴티티 (폰트, 컬러, 질감)
3. **폰트 제작·배포 전략 (NotoSans 포크, 다국어 피드, 민희진 벤치마킹)** ← 신규

3번이 추가되면서 findings.md가 상당히 커졌다. 만약 폰트 전략이 더 깊어지면 독립 프로젝트(`ck-font-strategy/`)로 분리하는 것을 검토해야 한다.

또한 민희진 벤치마킹이 매력적이지만, **CK와 오케이레코즈의 규모·리소스·산업이 완전히 다르다는 점**을 잊으면 안 된다. 벤치마킹에서 "구조적 원리"만 추출하고, 실행 수준의 모방은 경계해야 한다.

## 실행 가능성

| 항목 | 가능 여부 | 비고 |
|------|----------|------|
| NotoSans 포크 (파라미터 수준) | **가능** | 아이폰에서는 어려움 — PC 작업 필요. 휴가 시 또는 하진·미주에게 위임 |
| NotoSans 포크 (글리프 수정) | **휴가 시 가능** | Glyphs/FontForge 필요, PC 전용 |
| 다국어 피드 이미지 제작 | **부분 가능** | 텍스트 기반이므로 Canva 등 아이폰 앱으로 가능. 다만 CK Font이 완성되어야 함 |
| 다국어 번역 | **가능** | Claude/번역 도구로 처리 가능 |
| 폰트 배포 페이지 구축 | **휴가 시 가능** | GitHub Pages 또는 간단한 랜딩페이지 |
| 민희진식 "전 접점 동시 오픈" | **현실적으로 어려움** | 인스타+X+웹사이트 동시 론칭은 PC 작업 + 준비 기간 필요 |
| 인스타 계정 개설 | **즉시 가능** | 아이폰 |
| Phase 1 (비공개 전용 사용) | **즉시 가능** | 폰트 완성 후 CK 작업물에만 적용 |
