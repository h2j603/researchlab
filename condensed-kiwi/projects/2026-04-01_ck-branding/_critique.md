# 비평 — CK 브랜딩

## 핵심 주장
CK는 영어 퍼블리싱을 통해 해외 오디언스를 확보하고, 팀 인지도가 개인 멤버에게 수혜가 되는 구조를 만든다.

## 범위 경계

**다루는 것**:
- CK 인스타그램의 언어 전략 (영어 제1언어)
- 비주얼 아이덴티티 방향 (Negative/Positive 확정)
- 팀 → 개인 수혜 구조 설계
- 벤치마킹 대상 분석

**다루지 않는 것**:
- 개인 계정의 운영 전략 (각자의 영역)
- 상세 마케팅/그로스 전략 (CK는 에이전시가 아님)
- 수익화 모델 (아직 단계가 아님)

## 가장 약한 고리

1. **"팀 → 개인 수혜 구조"가 실제로 작동하려면 전제 조건이 있음.** CK 계정 자체가 먼저 콘텐츠 볼륨과 퀄리티를 갖춰야 트래픽이 발생한다. 계정 개설만으로는 아무 일도 일어나지 않음. 첫 3개월간 꾸준한 포스팅이 선행되어야 하는데, 혁의 군복무 제약 + 하진/미주의 참여 빈도가 이를 뒷받침할 수 있는지 미확인.

2. **영어 캡션의 퀄리티 관리.** 영어가 모국어가 아닌 팀이 영어로 퍼블리싱할 때, 어색한 영어는 오히려 프로페셔널한 인상을 해칠 수 있음. 해결책: 캡션을 최소화하고 비주얼로 승부 (Studio Feixen, NORM 모델), 또는 영어 네이티브 체크.

3. **하진, 미주의 동의 여부.** 영어 퍼블리싱은 혁의 방향인데, 팀원들이 이 전략에 동의하고 참여할 수 있는지 확인 필요. 캡션/해시태그 작성이 영어로 전환되면 하진·미주에게 추가 부담이 될 수 있음.

## 반론

**"영어 퍼블리싱이 해외 유입에 효과적이다"가 틀렸다면:**
- 인스타그램 그래픽 디자인 커뮤니티에서 언어는 부차적. **비주얼이 좋으면 언어와 무관하게 확산**됨. 한국어로 써도 해외에서 팔로우하는 디자인 계정은 많음 (일상의실천, 슬기와 민 등). 영어 전환이 유입에 결정적인 변수인지 불확실.
- 오히려 **한국어가 차별화 요소**가 될 수 있음. 한글 타이포그래피 자체가 해외 오디언스에게 시각적 흥미를 유발할 수 있는데, 영어로 전환하면 이 차별화가 사라짐.

**"팀 인지도가 개인에게 수혜가 된다"가 틀렸다면:**
- 팀 계정과 개인 계정의 성격이 너무 다르면 트래픽이 전환되지 않음. CK에서 보여주는 작업과 개인 계정의 작업이 연결성이 있어야 유입이 의미 있음.

## 복잡도 경고

현재 적정 수준이나, **브랜딩 리서치와 퍼블리싱 전략이 혼재**되기 시작함.
- 비주얼 아이덴티티 (로고, 컬러, 타이포) → 디자인 실무 영역
- 퍼블리싱 전략 (언어, 빈도, 포맷) → 운영 영역

이 둘이 하나의 findings.md에 있는 것은 괜찮지만, 각각이 비대해지면 분리 필요.

## 실행 가능성

| 항목 | 가능 여부 | 비고 |
|------|----------|------|
| 인스타 계정 개설 | **즉시 가능** | 아이폰으로 가능 |
| 영어 바이오 작성 | **즉시 가능** | |
| 첫 포스트 제작 | **가능** | 디지털 작업은 아이폰 앱으로도 가능하나 제약 있음 |
| 비주얼 아이덴티티 시안 | **부분 가능** | 혁은 컨셉/디렉션만, 시안 제작은 하진·미주에게 위임 |
| 피드 그리드 톤 통일 | **첫 포스트 이후** | 최소 6~9개 포스트가 쌓여야 그리드 효과가 보임 |
| 리소그래프 인쇄 테스트 | **휴가 시 가능** | 외주 인쇄소 이용 |
