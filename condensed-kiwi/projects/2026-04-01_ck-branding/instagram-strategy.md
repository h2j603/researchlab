# CK 인스타그램 퍼블리싱 전략 — 벤치마킹 & 초반 팀 공개

기존 `findings.md`의 §1(인스타그램 퍼블리싱 전략)을 확장하고, **초반 팀 공개 전략**과 **인스타그램 퍼스트 런칭 벤치마킹**을 추가 조사한 문서.

## 1. 인스타그램 퍼스트 런칭 벤치마킹

### 1.1 Jacquemus — 개인 서사 = 브랜드 서사

Simon Porte Jacquemus는 브랜드 계정과 개인 계정을 사실상 통합 운영한다. 전통적 광고 캠페인 없이, 유기적 소셜 성장에만 집중해서 글로벌 럭셔리 브랜드로 성장한 대표 사례.

**CK에 적용 가능한 구조**:
- 브랜드 계정에 멤버의 **개인적 취향과 일상**을 녹여내는 방식
- 완벽한 결과물 사진보다 **감정적 연결**을 유발하는 콘텐츠 우선
- 한정판 드롭(limited drops)으로 희소성 유지 — CK의 경우 zine, 포스터 등의 소량 에디션

**CK와의 차이점**: Jacquemus는 1인 브랜드. CK는 3인 팀이므로, 개인 서사를 팀 서사로 확장하는 변환이 필요. 누가 이 팀의 "얼굴"인가? 혹은 얼굴이 없는 것이 전략인가?

### 1.2 Fashion Nova — 마이크로 인플루언서 대군

인스타그램을 핵심 채널로 설정하고, 수천 명의 소규모 크리에이터와 협업하여 10억 달러 규모로 성장. 2020년 기준 1,182명의 크리에이터가 5,700개 이상의 스폰서 포스트 제작.

**CK 시사점**: CK 규모에서는 불가능한 모델이지만, 원리는 참고할 만함 — **다른 크리에이터가 CK의 작업물을 자발적으로 공유하게 만드는 구조**. 폰트 배포, 프린트 템플릿 공유 등이 이 역할을 할 수 있음.

### 1.3 디자인 스튜디오의 인스타그램 퍼스트 사례

| 스튜디오 | 전략 | CK 시사점 |
|----------|------|-----------|
| **Studio Feixen** | 비주얼이 곧 언어. 캡션 최소화, 월 4~5회 | CK의 기존 벤치마킹 대상. 비주얼 우선 전략 참고 |
| **Animade** | WIP + "브레이크다운"(제작 과정 분해) 공유 → 클라이언트가 인스타에서 직접 연락 | **프로세스 공개가 영업이 되는 구조** — CK에 직접 적용 가능 |
| **Design Bridge and Partners** | 스토리 + "Culture" 하이라이트로 팀/작업공간 아카이빙 | 팀 문화 콘텐츠의 하이라이트 활용법 |
| **Studio Mut** | 마일스톤 셀러브레이션 + 팀 크레딧 강조 | 팀 크레딧 부여 방식 참고 |
| **Pink Pony Creative** | 설립자 개성을 전면에 내세운 인격 기반 콘텐츠 | 소규모 팀의 "인격" 전략 |

**핵심 관찰**: 비하인드 콘텐츠의 스토리 완주율이 40%에 달한다는 데이터. Animade는 인스타그램 포스트를 본 클라이언트가 직접 연락해온 경험이 있음. **프로세스 콘텐츠 = 잠재 클라이언트 유입 경로**.

## 2. 초반 팀 공개(Team Reveal) 전략

### 2.1 팀 공개의 두 가지 접근법

**A. 작업물 선행 → 팀 후공개**
- 먼저 작업물로 "이건 뭐지?"라는 호기심을 유발
- 충분한 비주얼 정체성이 쌓인 후 팀을 공개
- 팀이 아니라 **결과물이 정체성**이 되는 구조
- 참고: Experimental Jetset (멤버보다 작업 스타일이 먼저 인지됨)

**B. 팀 선행 → 작업물 후공개**
- "우리는 이런 사람들이다"를 먼저 보여주고
- 그 사람들이 만드는 것에 대한 기대감 형성
- 참고: Jacquemus (사람이 먼저, 브랜드가 따라옴)

**C. 동시 공개 (CK 제안)**
- CK의 다국어 피드 론칭 캠페인이 **작업물이면서 동시에 팀 존재 선언**
- "Kiwi for sale"은 팀의 존재를 알리는 메시지 + CK Font의 쇼케이스
- 이 경우 팀 멤버 소개는 두 번째 단계에서 자연스럽게 진행

### 2.2 팀 리빌 콘텐츠 유형

| 콘텐츠 유형 | 설명 | CK 실행 가능성 |
|-------------|------|---------------|
| **프로세스 공유** | 스케치, 레이아웃, 폰트 파라미터 조정 과정 | **높음** — 디지털 스크린샷으로 가능 |
| **멤버 시리즈** | 한 명씩 소개하는 연속 포스트 | **중간** — 카메라 불가로 사진 사용 제한, 일러스트/타이포로 대체 가능 |
| **"무엇을 만드는가" 시리즈** | 각 멤버가 관심 있는 주제/레퍼런스 공유 | **높음** — 디지털 콘텐츠만으로 충분 |
| **비하인드 도구** | 사용하는 도구, 환경, 세팅 공유 | **중간** — 스크린 캡처는 가능, 물리적 환경 촬영 불가 |
| **마일스톤** | "첫 번째 포스트", "폰트 첫 글리프" 등 | **높음** — 의미 부여만으로 콘텐츠화 가능 |

### 2.3 CK 초반 팀 공개 시나리오 (제안)

**Phase 0 — 프리런칭 (계정 개설 전후)**
- 프로필 세팅: 영어 바이오, CK 로고/워드마크
- 첫 9개 포스트를 사전 기획 (그리드 효과를 위한 최소 단위)

**Phase 1 — 존재 선언 (1~2주차)**
- 다국어 피드 캠페인: "Kiwi for sale" 시리즈 (기존 findings.md § 2 전략)
- 이 단계에서 팀 멤버는 직접 노출되지 않음. **작업물이 먼저**
- CK Font의 다국어 구현이 콘텐츠 자체

**Phase 2 — 프로세스 공개 (3~4주차)**
- CK Font 제작 과정: 파라미터 조정 스크린샷, before/after
- 옵아트 패턴 실험 과정: 기하학 패턴 이터레이션
- 리소 질감 시뮬레이션: Multiply 블렌드 과정
- 캡션에 멤버 이름 태그 시작 (아직 멤버 소개는 아님)

**Phase 3 — 팀 공개 (5~6주차)**
- 각 멤버를 CK의 비주얼 언어로 소개하는 포스트
- 사진이 아닌 **타이포그래피 기반 소개**: CK Font로 이름/역할 표현
- 카메라 불가 제약을 역이용 — "인물 사진 없이 사람을 소개하는 방식" 자체가 CK의 미학적 선택

> **이 시나리오의 핵심**: 카메라 불가 제약을 약점이 아니라 **스타일로 전환**. 인물 사진 없이 팀을 소개하는 것이 CK의 "텍스트의 모듈화" + 타이포 중심 정체성과 일관.

## 3. 그리드 디자인 전략

### 3.1 2025년 그리드 포맷 변경

인스타그램이 2024년 말 1:1 정사각형에서 **3:4 세로형**으로 전환. CK는 이 새 포맷을 처음부터 반영해야 함.

**실행 원칙**:
- 핵심 시각 요소는 **1080×1080px 세이프 존**에 배치 (정사각형·세로형 양쪽 대응)
- CK의 다국어 피드는 텍스트 중심이므로, 세로 확장 공간에 여백 또는 부가 패턴 배치

### 3.2 CK에 적합한 그리드 전략

findings.md § 2.4의 "그리드 설치작품" 개념과 연결:

| 전략 | CK 적용 |
|------|---------|
| **체커보드** | 다국어 포스트 + 작업물 교대 → 규칙적이면서 다양한 시각 리듬 |
| **컬러 행(Row)** | 3개 포스트가 하나의 행 = 하나의 컬러 무드. CK의 고채도+그레이 조합을 행 단위로 운영 |
| **모자이크/파노라마** | 연속 3개 포스트로 하나의 큰 이미지 구성 — 옵아트 패턴에 효과적. 단, 3:4 전환으로 재설계 필요 |
| **시스템적 반복** | Sol LeWitt / On Kawara 방식 — 동일 구조, 다른 변수(언어, 색상). CK 다국어 피드의 본질 |

**제안: 시스템적 반복 + 컬러 행 병행**
- 다국어 피드 기간(Phase 1): 동일 레이아웃의 시스템적 반복
- 작업물 축적 기간(Phase 2~): 3개 포스트 = 1행의 컬러 스토리

### 3.3 그리드 사전 계획

- 최소 9개(3행) 포스트를 미리 기획한 후 첫 업로드
- Preview/Planoly 등의 그리드 미리보기 앱 활용 (아이폰 가능)
- 그리드를 **잡지 레이아웃**으로 생각: 각 행이 하나의 스프레드

## 4. 콘텐츠 필러(Pillar) 구조 제안

CK 인스타그램의 반복 콘텐츠 카테고리:

| 필러 | 비율 | 콘텐츠 예시 |
|------|------|------------|
| **완성 작업물** | 40% | 포스터, 타이포 실험, zine 표지 |
| **프로세스** | 25% | 스케치→완성 과정, 폰트 파라미터 조정, 패턴 이터레이션 |
| **레퍼런스 큐레이션** | 15% | 영감을 준 것들 (크레딧 필수). 캐러셀 포맷 |
| **팀/문화** | 10% | 멤버 소개, 작업 환경, 사용 도구 |
| **실험/놀이** | 10% | 완성되지 않은 시도, 실패한 시안, 변주 |

### 포스팅 빈도

기존 findings.md의 제안(주 2~3회)을 유지하되, **Phase 1(론칭 캠페인) 기간에는 매일 1포스트**로 집중 투하 가능. 다국어 피드는 미리 제작 가능하므로 예약 게시(인스타그램 내장 기능 또는 Creator Studio)로 운영.

## 5. 벤치마킹 비교 종합

### 5.1 CK와 유사한 포지션의 벤치마킹 대상

| 벤치마킹 대상 | 공통점 | 차이점 | CK가 취할 것 |
|--------------|--------|--------|-------------|
| **Jacquemus** | 인스타 퍼스트, 유기적 성장 | 1인 vs 3인 팀, 패션 vs 그래픽 | 감정적 연결 방식, 희소성 전략 |
| **Animade** | 소규모 크리에이티브 팀, 프로세스 공유 | 애니메이션 vs 그래픽/인쇄 | WIP 공유 → 클라이언트 유입 구조 |
| **Studio Feixen** | 기하학+고채도, 비주얼 언어가 곧 소통 | 3D/모션 vs 인쇄 질감 | 캡션 최소화, 비주얼 우선 |
| **Design Bridge** | 팀 문화 하이라이트 운영 | 대형 에이전시 vs 소규모 팀 | 하이라이트 아카이빙 구조 |
| **Hato Press** | 리소 중심, 출판+스튜디오 분리 운영 | 런던 기반 vs 한국 | 계정 분리 운영 모델 참고 |

### 5.2 CK만의 차별화 포인트

기존 벤치마킹 대상 중 **인물 사진 없이 팀을 운영하는 사례**는 거의 없다. 대부분의 "팀 리빌" 콘텐츠는 멤버 사진, 작업실 사진, 비하인드 영상에 의존한다.

CK의 카메라 불가 제약은 역설적으로:
1. **타이포그래피 순수주의** — 텍스트와 그래픽만으로 정체성을 구축하는 것이 CK의 비주얼 아이덴티티(컨덴스드 타이포 + 옵아트)와 정합
2. **미스터리/익명성** — 멤버의 얼굴이 보이지 않는 것이 오히려 브랜드 미학의 일부가 됨
3. **재현 가능성** — 사진에 의존하지 않으므로, 모든 콘텐츠가 디지털로 제작·수정 가능. 군복무 중 아이폰 작업과 양립
