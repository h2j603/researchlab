# CK 브랜딩 — 리서치 및 벤치마킹

---

## 1. 인스타그램 퍼블리싱 전략

### 1.1 1언어: 영어

CK의 인스타그램 퍼블리싱은 **영어를 제1언어로** 한다.

**결정 근거**:
- 한국 그래픽 디자인 시장의 풀이 좁음 — 국내만 타겟하면 도달 범위가 제한적
- 팀 단위로 해외 오디언스를 유입시키면, **팀의 인지도가 개인에게도 수혜**가 되는 구조를 만들 수 있음
- 팀 → 개인으로의 유입 경로: CK 계정에서 멤버 개인 계정(@hyuk.xyz 등)으로 트래픽 흐름
- 영어 퍼블리싱은 해외 스튜디오/기관/페어와의 접점을 넓히는 가장 저비용 수단

**실행 방식**:
- 포스트 캡션: 영어 기본. 한국어 병기 가능하되, 영어가 먼저
- 해시태그: 영어 중심 (#graphicdesign #typography #risograph #zine 등)
- 프로필 바이오: 영어
- 스토리/리얼스: 텍스트 오버레이는 영어

### 1.2 팀 → 개인 수혜 구조

```
[CK 인스타그램] ──해외 유입──→ [팀 인지도 상승]
        │                              │
        ├── 프로필에 멤버 태그 ──→ @hyuk.xyz
        ├── 프로필에 멤버 태그 ──→ @hajin (TBD)
        └── 프로필에 멤버 태그 ──→ @miju (TBD)
```

- 각 포스트에 참여 멤버를 태그/크레딧
- CK 프로필에 멤버 개인 계정 링크
- 개인 계정에서도 CK 작업을 리포스트/언급 → 양방향 트래픽

### 1.3 해외 타겟 전략

**벤치마킹 — 영어 퍼블리싱으로 해외 오디언스를 확보한 비영어권 스튜디오**:

| 스튜디오 | 국적 | 인스타 언어 | 팔로워 규모 | 특징 |
|----------|------|-------------|-------------|------|
| Studio Feixen | 스위스 | 영어 | 50K+ | 비주얼 중심, 캡션 최소화 |
| Experimental Jetset | 네덜란드 | 영어 | 100K+ | 일관된 톤, 프로세스 공유 |
| NORM | 스위스 | 영어 | 소규모 | 작업물 중심, 캡션 거의 없음 |
| Hato Press | 영국 | 영어 | 30K+ | 리소그래프 중심, 컬러풀 |
| Felipe Pantone | 스페인 | 영어 | 1M+ | 옵아트+디지털, 비주얼이 언어 자체 |

**관찰**: 비영어권 스튜디오들이 글로벌 인지도를 쌓은 패턴:
1. **비주얼이 곧 언어** — 캡션에 의존하지 않아도 작업이 전달됨
2. **영어 캡션은 간결하게** — 작업 설명, 크레딧, 맥락 정도
3. **해시태그 전략** — 영어 해시태그가 발견 가능성(discoverability)의 핵심
4. **일관된 피드 그리드** — 팀의 비주얼 랭귀지가 피드 자체에서 드러남

### 1.4 CK 인스타그램 운영 원칙 (제안)

1. **포스트 빈도**: 주 2~3회 (무리하지 않는 선에서 리듬 유지)
2. **콘텐츠 유형**:
   - 완성 작업물 (포스터, 타이포 실험, zine 등)
   - 프로세스 스냅샷 (스케치, 레이아웃 과정 — 카메라 불가 시 디지털 스크린샷)
   - 레퍼런스 큐레이션 (영감을 준 것들, 크레딧 필수)
3. **캡션 포맷**:
   ```
   [작업 제목] — [한 줄 설명]

   [크레딧: 디렉터, 참여 멤버]
   [멤버 태그]

   #condensedkiwi #graphicdesign [관련 태그]
   ```
4. **피드 비주얼 통일**: 브랜딩 확정 후, 피드 그리드의 톤 앤 매너 통일

## 2. 계정 운영 전략: CK 폰트 + 다국어 피드

### 2.1 전략 구조

NotoSans를 베이스로 파라미터를 조정해 CK 전용 폰트를 제작하고, 그 폰트로 "Kiwi for sale" 같은 문구를 n개 언어로 — 한 언어당 한 피드 — 인스타그램에 게시한다. 이후 CK 폰트를 배포한다.

```
[NotoSans 포크] → [파라미터 변형 = CK Font] → [다국어 피드 게시] → [폰트 배포]
     ↑ 베이스            ↑ 정체성 주입            ↑ 쇼케이스/그로스       ↑ 전염
```

### 2.2 민희진 오케이레코즈 벤치마킹

이 전략은 민희진의 크리에이티브 디렉팅 모델을 벤치마킹한다.

**민희진 모델의 작동 원리**:

1. **정반합(正反合) 전략**: 그 시대의 '정(正)'이 있으면, 그 반대를 제시하는 '반(反)'으로 신선함을 만든다. 민희진은 SM 시절부터 이 법칙으로 소녀시대(친근함 vs 기존 걸그룹의 비현실성), 뉴진스(자연스러움 vs K-POP의 과도한 세계관)를 디렉팅했다.
2. **세계관 부정을 통한 세계관 형성**: 민희진은 "K-POP이 말하는 세계관에 반감이 많다"고 공언했다. 그러나 일관된 비주얼 언어의 반복을 통해 자연스럽게 세계관이 형성되도록 설계한다. 선언이 아니라 축적.
3. **모든 접점의 통제**: 앨범 아트, MV, 머천다이즈, SNS — 모든 비주얼 터치포인트를 하나의 시각 언어로 통일. 오케이레코즈(ooak = One Of A Kind)도 출범 시 인스타그램, X, 홈페이지, 티저 영상을 동시에 열며 전 접점을 장악했다.
4. **1차 상품이 존재**: 민희진 모델에서 미학적 체계(비주얼 시스템)는 1차 상품(아이돌, 음악)을 감싸는 포장이다. 포장만으로는 욕망이 발생하지 않는다.

**CK에 대입**:

| 민희진 모델 | CK 적용 |
|---|---|
| 정반합 — 기존 관습의 반대 제시 | NotoSans(가장 보편적 폰트)를 변형해 고유한 것으로 만듦 — "보편성의 전유" |
| 세계관 부정 → 축적으로 형성 | CK 폰트 + 다국어 피드 + 리소 질감의 반복이 세계관을 만듦 |
| 모든 접점의 시각적 통제 | CK 폰트가 피드, 프로필, 작업물, 배포물 전체를 관통 |
| 1차 상품 존재 | **미정 — 아래 2.5에서 상세 논의** |

### 2.3 NotoSans 포크의 기술적·개념적 분석

**기술적 현실**:
- NotoSans는 Variable Font으로 제공됨. 조정 가능한 축: `wght`(100–900), `wdth`(62.5–100), `ital`(0–1)
- OFL(Open Font License) 라이선스 → 포크·변형·재배포 합법. 단, 폰트 단독 판매는 불가
- 1,000개 이상의 언어, 150개 이상의 문자 체계를 커버 → 다국어 피드 전략의 물리적 기반
- 빌드 소스가 GitHub에 공개되어 있어(notofonts 조직) 파라미터 수준을 넘어 글리프 레벨 수정도 가능

**개념적 레이어 — 이것이 단순한 폰트 커스터마이징이 아닌 이유**:

NotoSans는 Google이 "모든 언어에서 두부(豆腐, tofu = 미지원 글리프를 나타내는 □)가 나오지 않게 하자"는 목표로 만든 프로젝트다. 이름 자체가 "No more tofu"의 줄임말이다. 즉 NotoSans의 설계 철학은 **극단적 보편주의** — 어떤 언어도 배제하지 않겠다는 기업적 선언.

CK가 이것을 포크해서 "자기 것"으로 만드는 행위는:
- **보편적인 것에서 고유한 것을 추출하는 행위** — 모두를 위해 설계된 것을 특정 집단의 정체성으로 전유
- **민희진의 정반합과 구조적으로 동일** — NotoSans(正, 보편)에 대한 CK Font(反, 고유)의 제시
- Google의 "no tofu" 유니버설리즘을 CK의 미학적 도구로 전환

**핵심 리스크**: 파라미터 조정(weight, width 변경)만으로는 NotoSans의 그림자에서 벗어나기 어렵다. wght/wdth 축만 건드려서는 "이건 NotoSans잖아"라는 인식을 깰 수 없음. 이 리스크에 대한 대응 경로:

1. **글리프 레벨 수정** — 특정 글자의 형태를 변형 (e.g. a, g, k의 대안적 형태). 소스가 공개되어 있으므로 기술적으로 가능하지만, 타입 디자인 전문성 필요
2. **커스텀 축 추가** — Variable Font의 커스텀 디자인 축을 만들어 NotoSans에 없는 표현 범위를 부여
3. **시각적 차별화를 포기하고 맥락적 차별화에 집중** — CK Font의 정체성은 글자 형태가 아니라 "CK가 사용하는 폰트"라는 맥락에서 발생한다는 입장. Spotify Mix가 Circular(기존 폰트)와 극적으로 다르지 않아도 "스포티파이의 폰트"로 인식되는 것과 같은 논리

### 2.4 다국어 피드의 다층적 기능

"Kiwi for sale"(= "컨덴스드 키위를 구매할 사람이 있나?")를 n개 언어로 한 피드씩 게시하는 행위는 최소 네 가지 레이어에서 동시에 작동한다:

**레이어 1 — 폰트 스페시먼**: 각 포스트가 CK Font의 다국어 구현을 증명한다. 피드 전체가 타이포그래피 카탈로그.

**레이어 2 — 알고리즘 해킹**: 인스타그램 알고리즘은 언어를 기반으로 콘텐츠를 서빙한다. 아랍어 포스트는 아랍어권에, 일본어 포스트는 일본어권에 도달한다. 즉 n개 언어 = n개 언어권 오디언스에 동시 노출. 이건 그로스 전략이면서 동시에 개념적 작업이다.

**레이어 3 — 자기 참조적 메시지**: "Kiwi for sale"은 CK(Condensed Kiwi) 자체를 지칭한다. "컨덴스드 키위를 살 사람?"이라는 물음을 30개 언어로 던지는 것은 — 팀의 존재를 알리는 광고이면서, 동시에 "우리를 원하는 사람이 있는가?"라는 실존적 질문. 콘텐츠이면서 메타콘텐츠.

**레이어 4 — 그리드 설치작품**: 인스타그램 프로필의 그리드 뷰에서 이 포스트들은 개별이 아닌 전체로 읽힌다. 동일한 레이아웃, 동일한 폰트, 다른 문자 — 타이포그래피적 반복과 변주의 패턴. 이건 Sol LeWitt이나 On Kawara의 개념미술에 가깝다. 갤러리가 아니라 시스템.

### 2.5 1차 상품 문제 — CK에서 무엇이 욕망의 대상인가

민희진 모델의 핵심 전제: 미학적 체계는 1차 상품을 감싸는 포장이며, 포장만으로는 욕망이 발생하지 않는다.

CK에서 가능한 경로:

| 경로 | 1차 상품 | 폰트의 역할 | 난이도 | 민희진 모델 적합성 |
|------|---------|------------|--------|-------------------|
| A | CK 폰트 자체 | 상품 = 도구 | 높음 — 폰트 자체에 욕망을 만들어야 함 | 낮음 |
| B | CK의 디자인 작업물 (포스터, zine 등) | 작업물에 반복 등장하며 자연스럽게 인지 → 추후 배포 | 중간 | **가장 높음** |
| C | 다국어 피드 자체가 콘텐츠 | 콘텐츠의 물리적 구현 수단 | 중간 — 메시지의 힘에 의존 | 중간 |
| D | B + C 병행 | 작업물과 피드가 서로를 보강 | 중간 | 높음 |

**경로 B가 민희진 모델에 가장 부합한다.** 민희진의 비주얼 시스템은 아이돌(1차 상품)과 함께 출시된다. CK Font도 CK의 작업물(포스터, zine, 클라이언트 워크)에 일관되게 사용되면서 인지도를 쌓고, 충분히 축적된 후에 배포하는 것이 자연스러운 순서다.

다국어 피드는 이 과정의 **론칭 캠페인** 역할을 할 수 있다 — 계정 개설 직후, 작업물이 쌓이기 전에 피드를 채우면서 동시에 폰트를 노출시키는 전략.

### 2.6 폰트 배포 전략 — 통제와 개방의 설계

민희진 전략에서 빠지면 안 되는 것: **통제와 희소성**. 모든 시각 요소를 극도로 통제하고, 쉽게 접근 못 하게 만들어서 욕망을 키운다. 그런데 폰트 "배포"는 본질적으로 통제를 푸는 행위다.

**현재 폰트 배포 모델의 스펙트럼**:

| 모델 | 사례 | 통제 수준 | 브랜드 효과 |
|------|------|----------|------------|
| 완전 개방 (무료 배포) | Google Fonts, Inter | 없음 | 보편화 — 정체성 희석 |
| 제한적 무료 (가입/DM 요구) | 일부 인디 파운드리 | 낮음 | 관계 형성 |
| 유료 판매 | Commercial Type, Dinamo | 중간 | 전문성 인정 |
| 비공개 전용 | Spotify Mix, Netflix Sans | 최고 | 독점적 정체성 |

CK의 위치는 어디여야 하는가?

**제안: 단계적 개방 (Staged Release)**
1. **Phase 1 — 비공개 전용**: CK 작업물에만 사용. 피드에서 폰트가 반복 노출되지만 다운로드 불가. "이 폰트 뭐야?"라는 질문이 쌓이게 만듦
2. **Phase 2 — 제한적 배포**: DM 요청 또는 CK zine 구매자에게만 제공. 희소성 유지하면서 커뮤니티 형성
3. **Phase 3 — 공개 배포**: 충분한 인지도가 쌓인 후 공개. 이 시점에서 폰트는 "CK의 폰트"로 이미 맥락이 형성되어 있으므로, 배포해도 정체성이 희석되지 않음

**참고 — OFL 라이선스 제약**: NotoSans 포크인 경우 OFL 조건상 폰트 단독 판매는 불가. 그러나 무료 배포 + 제한적 접근(가입, 뉴스레터 구독 등)은 가능. 유료화하려면 NotoSans 베이스에서 충분히 벗어나 독자적 폰트로 인정받을 수 있는 수준의 변형이 필요.

## 3. 비주얼 아이덴티티 방향

### 3.1 확정된 Negative (배제)
- 화려한 세리프
- 크롬, 메탈 텍스처
- 핸드드로잉
- 흐물흐물한 타이포

### 3.2 확정된 Positive (추구)
- 산세리프, 컨덴스드 폰트, 볼드한 웨이트
- 종이로 오려붙인 텍스처
- 리소프린트 같은 질감
- 고채도(네온 계열) + 그레이 컬러 조합
- 텍스트의 모듈화
- 형광인쇄
- 기하학적으로 추상화한 셰입
- 옵아트

### 3.3 벤치마킹 대상별 인사이트

**Studio Feixen** — 기하학적 셰입 + 고채도 컬러의 조합. CK의 positive 방향과 가장 겹침이 큼. 다만 Feixen은 3D/모션이 강한데, CK는 인쇄 질감(리소, 콜라주)으로 차별화 가능.

**Karel Martens** — 리소/레터프레스 질감의 대가. 모노프린트 작업에서 보이는 잉크 레이어와 불완전한 겹침이 CK가 추구하는 "최신 기술로 최고의 아날로그함"과 맞닿아 있음.

**Hato Press** — 리소그래프 중심 출판. 컬러 팔레트와 인쇄 실험이 CK의 방향과 유사. 인스타그램 운영도 참고할 만함.

**Felipe Pantone** — 옵아트 + 디지털 글리치. CK의 옵아트 관심사와 연결되지만, Pantone은 극도로 디지털/글로시한 방향이라 CK와는 질감에서 분기.
