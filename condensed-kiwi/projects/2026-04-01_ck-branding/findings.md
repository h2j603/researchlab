# CK 브랜딩 — 리서치 및 벤치마킹

---

## 1. 인스타그램 퍼블리싱 전략

### 1.1 1언어: 영어

CK의 인스타그램 퍼블리싱은 **영어를 제1언어로** 한다.

**결정 근거**:
- 한국 그래픽 디자인 시장의 풀이 좁음 — 국내만 타겟하면 도달 범위가 제한적
- 팀 단위로 해외 오디언스를 유입시키면, **팀의 인지도가 개인에게도 수혜**가 되는 구조를 만들 수 있음
- 팀 → 개인으로의 유입 경로: CK 계정에서 멤버 개인 계정(@hyuk.xyz 등)으로 트래픽 흐름
- 영어 퍼블리싱은 해외 스튜디오/기관/페어와의 접점을 넓히는 가장 저비용 수단

**실행 방식**:
- 포스트 캡션: 영어 기본. 한국어 병기 가능하되, 영어가 먼저
- 해시태그: 영어 중심 (#graphicdesign #typography #risograph #zine 등)
- 프로필 바이오: 영어
- 스토리/리얼스: 텍스트 오버레이는 영어

### 1.2 팀 → 개인 수혜 구조

```
[CK 인스타그램] ──해외 유입──→ [팀 인지도 상승]
        │                              │
        ├── 프로필에 멤버 태그 ──→ @hyuk.xyz
        ├── 프로필에 멤버 태그 ──→ @hajin (TBD)
        └── 프로필에 멤버 태그 ──→ @miju (TBD)
```

- 각 포스트에 참여 멤버를 태그/크레딧
- CK 프로필에 멤버 개인 계정 링크
- 개인 계정에서도 CK 작업을 리포스트/언급 → 양방향 트래픽

### 1.3 해외 타겟 전략

**벤치마킹 — 영어 퍼블리싱으로 해외 오디언스를 확보한 비영어권 스튜디오**:

| 스튜디오 | 국적 | 인스타 | 팔로워 | 특징 |
|----------|------|--------|--------|------|
| Studio Feixen | 스위스 | 영어 | 213K | 기하학 셰입 + 고채도, 월 4~5회 포스팅, 비주얼 놀이터 |
| Experimental Jetset | 네덜란드 | 영어 | 84K | "Turning language into objects", 모더니즘/Helvetica 일관 |
| Studio fnt | 한국 | 영어 병기 | 57K | 한글 타이포의 국제적 매력, CommArts·It's Nice That 피처 |
| Nendo | 일본 | 영어 | 211K | 극도 절제(474개 포스트), 양보다 질, 1계정만 팔로우 |
| Groovisions | 일본 | 영어 | 소규모 | "Chappie" 캐릭터 시스템으로 언어 장벽 돌파 |
| Hato Press | 영국 | 영어 | 66K | 리소그래프 출판 + 워크숍, @hatopress + @studiohato 분리 운영 |
| Felipe Pantone | 스페인 | 영어 | 1M+ | 옵아트+디지털 글리치, 비주얼이 언어 자체 |

**관찰 — 글로벌 인지도를 쌓은 공통 패턴**:
1. **비주얼이 곧 언어** — 캡션에 의존하지 않아도 작업이 전달됨
2. **영어 캡션은 간결하게** — 작업 제목, 클라이언트명, 연도 등 최소 정보만
3. **해시태그 절제** — 과도한 해시태그 지양, 브랜드 태그 + 최소 카테고리만
4. **일관된 피드 그리드** — 팀의 비주얼 랭귀지가 피드 자체에서 드러남
5. **영어권 미디어 노출이 핵심 유입 경로** — It's Nice That, Dezeen, CommArts에 피처되는 것이 팔로워 유입의 결정적 계기
6. **이중 언어 캡션은 발견 가능성(discoverability)을 2배로 높임** — VeraContent 리서치 기반

### 1.4 CK 인스타그램 운영 원칙 (제안)

1. **포스트 빈도**: 주 2~3회 (무리하지 않는 선에서 리듬 유지)
2. **콘텐츠 유형**:
   - 완성 작업물 (포스터, 타이포 실험, zine 등)
   - 프로세스 스냅샷 (스케치, 레이아웃 과정 — 카메라 불가 시 디지털 스크린샷)
   - 레퍼런스 큐레이션 (영감을 준 것들, 크레딧 필수)
3. **캡션 포맷**:
   ```
   [작업 제목] — [한 줄 설명]

   [크레딧: 디렉터, 참여 멤버]
   [멤버 태그]

   #condensedkiwi #graphicdesign [관련 태그]
   ```
4. **피드 비주얼 통일**: 브랜딩 확정 후, 피드 그리드의 톤 앤 매너 통일

## 2. 계정 운영 전략: CK 폰트 + 다국어 피드

### 2.1 전략 구조

NotoSans를 베이스로 파라미터를 조정해 CK 전용 폰트를 제작하고, 그 폰트로 "Kiwi for sale" 같은 문구를 n개 언어로 — 한 언어당 한 피드 — 인스타그램에 게시한다. 이후 CK 폰트를 배포한다.

```
[NotoSans 포크] → [파라미터 변형 = CK Font] → [다국어 피드 게시] → [폰트 배포]
     ↑ 베이스            ↑ 정체성 주입            ↑ 쇼케이스/그로스       ↑ 전염
```

### 2.2 민희진 오케이레코즈 벤치마킹

이 전략은 민희진의 크리에이티브 디렉팅 모델을 벤치마킹한다.

**민희진 모델의 작동 원리**:

1. **정반합(正反合) 전략**: 그 시대의 '정(正)'이 있으면, 그 반대를 제시하는 '반(反)'으로 신선함을 만든다. 민희진은 SM 시절부터 이 법칙으로 소녀시대(친근함 vs 기존 걸그룹의 비현실성), 뉴진스(자연스러움 vs K-POP의 과도한 세계관)를 디렉팅했다.
2. **세계관 부정을 통한 세계관 형성**: 민희진은 "K-POP이 말하는 세계관에 반감이 많다"고 공언했다. 그러나 일관된 비주얼 언어의 반복을 통해 자연스럽게 세계관이 형성되도록 설계한다. 선언이 아니라 축적.
3. **모든 접점의 통제**: 앨범 아트, MV, 머천다이즈, SNS — 모든 비주얼 터치포인트를 하나의 시각 언어로 통일. 오케이레코즈(ooak = One Of A Kind)도 출범 시 인스타그램, X, 홈페이지, 티저 영상을 동시에 열며 전 접점을 장악했다.
4. **1차 상품이 존재**: 민희진 모델에서 미학적 체계(비주얼 시스템)는 1차 상품(아이돌, 음악)을 감싸는 포장이다. 포장만으로는 욕망이 발생하지 않는다.

**CK에 대입**:

| 민희진 모델 | CK 적용 |
|---|---|
| 정반합 — 기존 관습의 반대 제시 | NotoSans(가장 보편적 폰트)를 변형해 고유한 것으로 만듦 — "보편성의 전유" |
| 세계관 부정 → 축적으로 형성 | CK 폰트 + 다국어 피드 + 리소 질감의 반복이 세계관을 만듦 |
| 모든 접점의 시각적 통제 | CK 폰트가 피드, 프로필, 작업물, 배포물 전체를 관통 |
| 1차 상품 존재 | **미정 — 아래 2.5에서 상세 논의** |

### 2.3 NotoSans 포크의 기술적·개념적 분석

**기술적 현실**:
- NotoSans는 Variable Font으로 제공됨. 조정 가능한 축: `wght`(100–900), `wdth`(62.5–100), `ital`(0–1)
- OFL(Open Font License) 라이선스 → 포크·변형·재배포 합법. 단, 폰트 단독 판매는 불가
- 1,000개 이상의 언어, 150개 이상의 문자 체계를 커버 → 다국어 피드 전략의 물리적 기반
- 빌드 소스가 GitHub에 공개되어 있어(notofonts 조직) 파라미터 수준을 넘어 글리프 레벨 수정도 가능

**개념적 레이어 — 이것이 단순한 폰트 커스터마이징이 아닌 이유**:

NotoSans는 Google이 "모든 언어에서 두부(豆腐, tofu = 미지원 글리프를 나타내는 □)가 나오지 않게 하자"는 목표로 만든 프로젝트다. 이름 자체가 "No more tofu"의 줄임말이다. 즉 NotoSans의 설계 철학은 **극단적 보편주의** — 어떤 언어도 배제하지 않겠다는 기업적 선언.

CK가 이것을 포크해서 "자기 것"으로 만드는 행위는:
- **보편적인 것에서 고유한 것을 추출하는 행위** — 모두를 위해 설계된 것을 특정 집단의 정체성으로 전유
- **민희진의 정반합과 구조적으로 동일** — NotoSans(正, 보편)에 대한 CK Font(反, 고유)의 제시
- Google의 "no tofu" 유니버설리즘을 CK의 미학적 도구로 전환

**핵심 리스크**: 파라미터 조정(weight, width 변경)만으로는 NotoSans의 그림자에서 벗어나기 어렵다. wght/wdth 축만 건드려서는 "이건 NotoSans잖아"라는 인식을 깰 수 없음. 이 리스크에 대한 대응 경로:

1. **글리프 레벨 수정** — 특정 글자의 형태를 변형 (e.g. a, g, k의 대안적 형태). 소스가 공개되어 있으므로 기술적으로 가능하지만, 타입 디자인 전문성 필요
2. **커스텀 축 추가** — Variable Font의 커스텀 디자인 축을 만들어 NotoSans에 없는 표현 범위를 부여
3. **시각적 차별화를 포기하고 맥락적 차별화에 집중** — CK Font의 정체성은 글자 형태가 아니라 "CK가 사용하는 폰트"라는 맥락에서 발생한다는 입장. Spotify Mix가 Circular(기존 폰트)와 극적으로 다르지 않아도 "스포티파이의 폰트"로 인식되는 것과 같은 논리

### 2.4 다국어 피드의 다층적 기능

"Kiwi for sale"(= "컨덴스드 키위를 구매할 사람이 있나?")를 n개 언어로 한 피드씩 게시하는 행위는 최소 네 가지 레이어에서 동시에 작동한다:

**레이어 1 — 폰트 스페시먼**: 각 포스트가 CK Font의 다국어 구현을 증명한다. 피드 전체가 타이포그래피 카탈로그.

**레이어 2 — 알고리즘 해킹**: 인스타그램 알고리즘은 언어를 기반으로 콘텐츠를 서빙한다. 아랍어 포스트는 아랍어권에, 일본어 포스트는 일본어권에 도달한다. 즉 n개 언어 = n개 언어권 오디언스에 동시 노출. 이건 그로스 전략이면서 동시에 개념적 작업이다.

**레이어 3 — 자기 참조적 메시지**: "Kiwi for sale"은 CK(Condensed Kiwi) 자체를 지칭한다. "컨덴스드 키위를 살 사람?"이라는 물음을 30개 언어로 던지는 것은 — 팀의 존재를 알리는 광고이면서, 동시에 "우리를 원하는 사람이 있는가?"라는 실존적 질문. 콘텐츠이면서 메타콘텐츠.

**레이어 4 — 그리드 설치작품**: 인스타그램 프로필의 그리드 뷰에서 이 포스트들은 개별이 아닌 전체로 읽힌다. 동일한 레이아웃, 동일한 폰트, 다른 문자 — 타이포그래피적 반복과 변주의 패턴. 이건 Sol LeWitt이나 On Kawara의 개념미술에 가깝다. 갤러리가 아니라 시스템.

### 2.5 1차 상품 문제 — CK에서 무엇이 욕망의 대상인가

민희진 모델의 핵심 전제: 미학적 체계는 1차 상품을 감싸는 포장이며, 포장만으로는 욕망이 발생하지 않는다.

CK에서 가능한 경로:

| 경로 | 1차 상품 | 폰트의 역할 | 난이도 | 민희진 모델 적합성 |
|------|---------|------------|--------|-------------------|
| A | CK 폰트 자체 | 상품 = 도구 | 높음 — 폰트 자체에 욕망을 만들어야 함 | 낮음 |
| B | CK의 디자인 작업물 (포스터, zine 등) | 작업물에 반복 등장하며 자연스럽게 인지 → 추후 배포 | 중간 | **가장 높음** |
| C | 다국어 피드 자체가 콘텐츠 | 콘텐츠의 물리적 구현 수단 | 중간 — 메시지의 힘에 의존 | 중간 |
| D | B + C 병행 | 작업물과 피드가 서로를 보강 | 중간 | 높음 |

**경로 B가 민희진 모델에 가장 부합한다.** 민희진의 비주얼 시스템은 아이돌(1차 상품)과 함께 출시된다. CK Font도 CK의 작업물(포스터, zine, 클라이언트 워크)에 일관되게 사용되면서 인지도를 쌓고, 충분히 축적된 후에 배포하는 것이 자연스러운 순서다.

다국어 피드는 이 과정의 **론칭 캠페인** 역할을 할 수 있다 — 계정 개설 직후, 작업물이 쌓이기 전에 피드를 채우면서 동시에 폰트를 노출시키는 전략.

### 2.6 폰트 배포 전략 — 통제와 개방의 설계

민희진 전략에서 빠지면 안 되는 것: **통제와 희소성**. 모든 시각 요소를 극도로 통제하고, 쉽게 접근 못 하게 만들어서 욕망을 키운다. 그런데 폰트 "배포"는 본질적으로 통제를 푸는 행위다.

**현재 폰트 배포 모델의 스펙트럼**:

| 모델 | 사례 | 통제 수준 | 브랜드 효과 |
|------|------|----------|------------|
| 완전 개방 (무료 배포) | Google Fonts, Inter | 없음 | 보편화 — 정체성 희석 |
| 제한적 무료 (가입/DM 요구) | 일부 인디 파운드리 | 낮음 | 관계 형성 |
| 유료 판매 | Commercial Type, Dinamo | 중간 | 전문성 인정 |
| 비공개 전용 | Spotify Mix, Netflix Sans | 최고 | 독점적 정체성 |

CK의 위치는 어디여야 하는가?

**제안: 단계적 개방 (Staged Release)**
1. **Phase 1 — 비공개 전용**: CK 작업물에만 사용. 피드에서 폰트가 반복 노출되지만 다운로드 불가. "이 폰트 뭐야?"라는 질문이 쌓이게 만듦
2. **Phase 2 — 제한적 배포**: DM 요청 또는 CK zine 구매자에게만 제공. 희소성 유지하면서 커뮤니티 형성
3. **Phase 3 — 공개 배포**: 충분한 인지도가 쌓인 후 공개. 이 시점에서 폰트는 "CK의 폰트"로 이미 맥락이 형성되어 있으므로, 배포해도 정체성이 희석되지 않음

**참고 — OFL 라이선스 제약**: NotoSans 포크인 경우 OFL 조건상 폰트 단독 판매는 불가. 그러나 무료 배포 + 제한적 접근(가입, 뉴스레터 구독 등)은 가능. 유료화하려면 NotoSans 베이스에서 충분히 벗어나 독자적 폰트로 인정받을 수 있는 수준의 변형이 필요.

## 3. 비주얼 아이덴티티 방향

### 3.1 확정된 Negative (배제)
- 화려한 세리프
- 크롬, 메탈 텍스처
- 핸드드로잉
- 흐물흐물한 타이포

### 3.2 확정된 Positive (추구)
- 산세리프, 컨덴스드 폰트, 볼드한 웨이트
- 종이로 오려붙인 텍스처
- 리소프린트 같은 질감
- 고채도(네온 계열) + 그레이 컬러 조합
- 텍스트의 모듈화
- 형광인쇄
- 기하학적으로 추상화한 셰입
- 옵아트

### 3.3 벤치마킹 대상별 심층 인사이트

#### Studio Feixen (스위스, 루체른)

기하학적 셰입 + 고채도 컬러의 조합. CK의 positive 방향과 **가장 겹침이 큼**.

- **철학**: "구체적으로 아무것에도 집중하지 않는다" — 스위스 디자인 전통에 뿌리를 두되 그 절제와 중립성을 의식적으로 거부
- **비주얼**: 고채도 볼드 컬러, 실험적 타이포(가변 폰트 중심), 추상 기하학적 셰입(곡선+각의 대비), 키네틱/애니메이션이 정체성의 핵심
- **최근 프로젝트**: Crocodile Chair(Phantom Hands, 2024), Westbund Art Fair 조각적 타이포 시스템(2024), MOV 서체(2025)
- **클라이언트**: Nike, Google, Chanel, NYT, Samsung, MIT 저널, Wired
- **CK와의 차별화**: Feixen은 3D/모션/디지털이 강함 → CK는 **인쇄 질감(리소, 콜라주)**으로 분기

#### Karel Martens (네덜란드)

리소/레터프레스 질감의 대가. CK가 추구하는 "최신 기술로 최고의 아날로그함"의 원형.

- **모노프린트 기법**: 버려진 금속 조각(자동차 부품 등)을 잉크 판으로, 파운드 페이퍼에 소형 제본 프레스로 수작업 인쇄. 하루 한 색, 건조 후 다음 색 → 한 프린트에 수일~수년 소요
- **철학**: "제약을 활용할 방법은 무수히 많다", "디자인은 사고방식의 결과여야 한다", "부재(absence)가 중요하다"
- **Stedelijk 회고전 "Unbound" (2025)**: 300점 이상, 65년 커리어 첫 대규모 회고전. Crouwel·Van Toorn과 함께 네덜란드 그래픽 디자인 '빅3'
- **최근 출판물**: 2026 일력(365개 모노프린트 컴포지션), *Love Letters*(2024, 1960년대 연인에게 보낸 봉투 디자인)
- **CK 시사점**: 잉크 레이어와 불완전한 겹침 → 디지털 도구로 이 "통제된 우연"을 재현하는 것이 CK의 과제

#### Hato Press (영국, 런던)

리소그래프 중심 출판 + 스튜디오 + 교육의 통합 모델.

- **설립**: 2009년, CSM 출신 Jackson Lam & Ken Kirton. "아티스트에게 인쇄를 접근 가능하게"
- **4-파트 모델**: Hato Press(출판) + Hato Studio(디자인) + Hato Labo(디지털/인터랙티브) + Learn with Hato(교육)
- **성과**: 50번째 아티스트 진 달성(2024), A6 리소 포맷으로 Takashi Homma, Jean Jullien, Daniel Eatock 등과 협업
- **인스타**: @hatopress(66K) + @studiohato(23K) + @hatostore__ 분리 운영
- **CK 시사점**: 출판 수익보다 워크숍·이벤트·스튜디오가 인쇄소를 지탱하는 구조 → CK도 인쇄 중심이라면 단독 수익원이 아닌 복합 모델 고려

#### Felipe Pantone (스페인/아르헨티나)

옵아트 + 디지털 글리치. CK의 옵아트 관심사와 연결되나 질감에서 분기.

- Pantone은 극도로 디지털/글로시/3D 방향 → CK는 **옵아트의 패턴을 리소/인쇄 질감으로 재해석**하는 것으로 차별화

## 4. 옵아트의 브랜딩 적용 가능성

### 4.1 옵아트가 브랜딩에 적합한 이유

- 기하학적 패턴의 수학적 설계, 시각적 착시, 반복 구조가 **"주목을 끌고 오래 기억되는" 브랜딩**에 적합
- 사용 빈도가 낮아 **차별화 효과가 큼**
- "혁신적이고 틀을 깨는" 브랜드 이미지 전달 가능

### 4.2 옵아트 브랜딩 적용 사례

| 사례 | 적용 방식 |
|------|----------|
| Halo Brewery (Underline Studio) | 크래프트 맥주 라벨에 옵아트 패턴 시스템 적용 |
| Sonos 로고 | 모아레 패턴 + 헤링 착시 + 환각적 움직임을 브랜드 마크에 결합 |
| 와인 라벨 (Meeta Panesar) | 평면이 3D처럼 보이는 옵티컬 기하학 라벨 |

### 4.3 옵아트 + 리소/실크스크린: CK에게 핵심인 조합

리소그래프는 원리상 실크스크린에 가깝다(단색 잉크를 스텐실로 찍는 방식). 옵아트의 **레이어 기반 패턴 구성**과 기술적으로 잘 맞는다.

> 옵아트의 반복 기하학 패턴은 리소의 색판 분리(color separation) 과정에서 자연스러운 **위치 어긋남(misregistration)**이 발생하면, 의도치 않은 **시각적 진동(visual vibration)**을 만들어냄. 디지털에서 재현할 수 없는 물성.

이것이 CK가 추구하는 **"최신 기술로 최고의 아날로그함"**과 직결된다. 디지털로 옵아트 패턴을 설계하되, 리소 인쇄의 물리적 불완전함이 최종 결과물에 고유한 질감을 더하는 구조.

### 4.4 모듈형 아이덴티티와 옵아트

CK의 positive 리스트에 "텍스트의 모듈화"가 있다. 옵아트의 반복 기하학 패턴은 **본질적으로 모듈적**이어서 모듈형 브랜드 아이덴티티와 자연스럽게 결합된다.

- 시각 요소를 결합·재조합하여 다양한 터치포인트에 적응하는 유연한 시스템
- 2025-2026년 "다이내믹 아이덴티티", "키네틱 타이포", "모션 로고" 트렌드와 연결
- Counter-Print 출판 *Optic*: 그래픽 디자인에서의 옵티컬 효과를 Periodic Structures, Interrupted Systems, Impossible Objects, Illusory Contours 등으로 분류

### 4.5 2025-2026 트렌드에서의 옵아트 부활 징후

"옵아트"라는 용어가 직접 쓰이지는 않지만, 옵아트 DNA를 가진 방향들이 부상 중:

- **도트 디스플레이/하프톤**: 반복 패턴으로 시각적 진동을 만드는 레트로-테크
- **기하학적 도형의 부활**: 미드센추리 회귀와 함께 대담한 기하학 패턴 증가
- **고대비 색상**: "bold" 검색 51%↑, "electric" 49%↑
- **키네틱 디자인**: 모션이 브랜딩의 핵심 요소로 격상
- **레트로-퓨처리즘**: 복고와 미래의 융합에서 옵아트적 시각 실험 등장

## 5. 리소그래프 + 컨덴스드 타이포 브랜딩 트렌드

### 5.1 리소그래프의 브랜딩 가치

입자감·오정렬·잉크 번짐 등 **"불완전함"이 차별화 요소**. 동시에:
- **친환경**: 대두 잉크, 저에너지
- **소량 인쇄 비용 효율성**: 10~500부 사이에서 최적
- 주요 리소 스튜디오: Ditto Press(런던), Risotto Studio(글래스고), Copy Corner(도쿄, 2024 개관)

### 5.2 컨덴스드 볼드 산세리프 트렌드

CK의 positive에 부합하는 서체들이 브랜딩 핵심 서체로 부상:

| 서체 | 특성 | 활용 맥락 |
|------|------|----------|
| Druk (Commercial Type) | 극단적 컨덴스드 볼드 | 헤드라인, 포스터 |
| GT Walsheim | 뉴트럴 산세리프 | 브랜딩 시스템 |
| Monument Extended | 확장형 볼드 | 대담한 타이틀 |
| Replica | 인더스트리얼 | 에디토리얼 |

**핵심**: 컨덴스드 서체는 압축·긴급함·모더니티를 전달. 헤드라인 전용으로 쓰고 여백과 조합하는 것이 원칙.

### 5.3 디지털 하이브리드 기법 (리소 질감의 디지털 구현)

군복무 중 아이폰 작업이 주인 혁에게 특히 중요한 섹션:

- **Multiply 블렌드 모드**: 레이어 분리 → 색판별 오프셋 → 리소 겹침 효과
- **레지스트레이션 오프셋**: 각 색판을 1~3px 어긋나게 배치 → 미세한 misregistration
- **Diffusion Dither 비트맵 변환**: 이미지를 비트맵으로 변환 → 리소 특유의 입자감
- **에셋/플러그인**: RetroSupply Kit, Procreate 리소 브러시 등

### 5.4 형광/네온 컬러 브랜딩

CK의 positive "고채도(네온 계열) + 그레이 컬러 조합"과 직결:

- 서브컬처의 색 언어가 메인스트림으로 확산 (Spotify 네온 그린, Nike, Maybelline)
- 인쇄 시 **Pantone 형광 잉크(801~807)** 필수
- 원칙: 어두운 배경 + 형광 1~2색 악센트

### 5.5 반작용 징후

> 리소가 이미 "세련된 대안"이 되면서, 일부는 일반 프린터의 더 날것의 흑백 미학으로 이동하는 반작용 조짐 (2026)

CK가 리소를 택한다면, "리소 = 힙하다"의 소비적 사용이 아닌 **기법적·개념적 깊이**로 차별화해야 함.
