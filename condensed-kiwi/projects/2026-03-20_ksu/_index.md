# AKU (Anti Kiwi Union, 구 KSU) 프로젝트

## 개요

CK(Condensed Kiwi) 세계관 내에서 **웹 키위바이러스에 대항하기 위해 결성된 가상 단체**. "진지한 태도로 헛소리를 하는" 전위적 프로젝트. CK=키위음모세력(AI키위제너레이터를 만들고 사라진 존재) ↔ AKU=키위에 대항하는 단체.

**기획**: 미주
**역할 분담**: 미주(기획/전체 브랜딩), 혁(웹), 하진(메뉴얼북) ← 3/25 이후 산출물 변경으로 재조정 필요
**목표 일정**: ~3월말 기획 / 4월 작업 / 5월 중순 쇼오프

## 현재 상태

- **KSU → AKU (Anti Kiwi Union, 아쿠)** 개명 확정 (3/25)
- **결성 계기: 웹 키위바이러스**로 합의 (3/25)
- 톤: B급컬트가 아닌 **나폴리탄** 방향 확인 (3/22)
- 세계관 구도: AI키위제너레이터(CK) ↔ AKU 대립
- CK(컨키) = 세계관 내 키위음모세력. "픽션 필터를 씌운" 존재
- 산출물: **인스타 게시물 중심 퍼블리싱** + 위키(별도) 방향 (3/25)
- 무드보드 제작 중 (노션, 현대미술 레퍼런스 위주)
- 웹 키위바이러스의 구체적 정의는 아직 미확정

## 산출물 (예정)

- **인스타그램 게시물** (메인) — 80~90% 완성도로 빠르게 양산, 한 피드 한줄 채우기
- 위키 (별도 운영) — 데이터 형식으로 세계관 구축
- AI 생성 인물 이미지 — 품남/외국인 방향, 인물설정 자세히
- 기관 브랜딩 (AKU)
- 기타(@)

## 핵심 미해결 질문

1. ~~KSU는 ___를 하는 단체이다.~~ → **AKU는 웹 키위바이러스에 대항하기 위해 결성된 단체** (3/25 합의)
2. **웹 키위바이러스란 구체적으로 무엇인가?** (어떻게 감염, 증상, 어디서 퍼지는지 미정)
3. 타겟 오디언스는 누구인가?
4. ~~CK 본체와의 관계~~ → CK=키위음모세력, AI키위제너레이터 ↔ AKU 대립 (3/25 합의). 구체적 스토리라인 미정
5. 위키는 누가 만드는가? (혁은 군복무 중 코딩 불가)
