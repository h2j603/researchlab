# Status — 리서치랩 전체 현황 파악

리서치랩의 전체 현황을 한눈에 보여준다.

## 실행 절차

1. 다음 항목을 조사하여 현황 리포트를 생성한다:

### inbox 현황
- `inbox/` 파일 수와 목록
- 가장 오래된 메모 날짜 (오래 방치된 것 강조)

### 진행 중인 프로젝트
- `projects/` 내 폴더 목록
- 각 프로젝트의 `status`와 마지막 수정일(`updated`)
- 오래 멈춰있는 프로젝트 표시

### 관심 분야
- `areas/` 내 파일/폴더 목록과 최근 활동

### 참고 자료
- `resources/` 내 파일 수
- 최근 추가된 자료

### 아카이브
- `archive/` 내 완료된 프로젝트 수

### 태그 현황
- 가장 많이 사용된 태그 Top 5
- 사용되지 않은 공식 태그 목록

### 연결 현황
- `related` 필드가 비어있는 노트 수
- 고립된 노트 (다른 노트에서 링크되지 않은 노트)

2. 현황을 깔끔한 표와 요약으로 정리하여 보여준다
3. 필요한 정리 작업이 있으면 제안한다 (예: inbox 정리, 오래된 프로젝트 아카이브 등)

## 주의사항

- 읽기 전용 명령어이다. 파일을 수정하지 않는다
- 간결하고 한눈에 파악 가능한 형태로 보여준다
