# Concept — 개념 정리 노트 작성

특정 개념, 용어, 이론을 체계적으로 정리한다.

## 실행 절차

1. 사용자의 입력(`$ARGUMENTS`)에서 정리할 개념을 파악한다
2. `resources/` 디렉토리에 `concept_개념명.md` 파일을 생성한다 (개념명은 kebab-case 영문)
3. `templates/concept-note-template.md` 템플릿을 따른다
4. 다음 항목을 채운다:
   - 정의 (간결하게)
   - 상세 설명
   - 예시 (구체적, 가능하면 시각 문화/디자인 맥락 포함)
   - 관련 개념 (기존 노트가 있으면 링크)
   - 출처
5. `_tags.md`에서 적절한 태그를 선택한다
6. `confidence` 레벨을 내용의 확실성에 맞게 설정한다

## 입력 예시

- `/concept 스큐어모피즘(Skeuomorphism)`
- `/concept 기호학에서의 도상(icon), 지표(index), 상징(symbol)`

## 주의사항

- 주장과 사실을 명확히 구분한다
- 사용자가 추상적 개념을 분류·체계화하는 것을 좋아하므로, 가능하면 분류 구조를 포함한다
