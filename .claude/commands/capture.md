# Capture — inbox에 아이디어 빠르게 캡처

사용자가 떠올린 아이디어, 질문, 메모를 `inbox/`에 빠르게 기록한다.

## 실행 절차

1. 사용자의 입력(`$ARGUMENTS`)을 바탕으로 핵심 내용을 파악한다
2. 파일명은 `inbox/YYYY-MM-DD_키워드.md` 형식으로 생성한다 (오늘 날짜 사용)
3. `templates/inbox-template.md` 템플릿을 따른다
4. 프론트매터의 `status`는 반드시 `inbox`, `tags`는 내용에 맞게 `_tags.md`에서 선택한다
5. 본문은 사용자가 말한 내용을 간결하게 정리한다. 과도하게 부풀리지 않는다
6. 같은 날짜에 같은 키워드 파일이 있으면 번호를 붙인다 (예: `_2.md`)
7. 파일 생성 후 경로와 한 줄 요약을 알려준다

## 입력 예시

- `/capture 요즘 카세트테이프가 다시 유행하는 현상이 흥미롭다`
- `/capture 브루탈리즘 건축과 웹 브루탈리즘의 관계?`

## 주의사항

- 최소한의 정리만 한다. inbox는 빠른 캡처가 목적이다
- 사용자에게 추가 질문하지 않고 바로 생성한다
