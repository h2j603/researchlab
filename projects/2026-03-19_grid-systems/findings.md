# 그리드 시스템 — 조사 내용

## 1. 교차점 배치 vs. 칸 배치: 사실 정리

### 1.1 보드게임 비교표

| 게임 | 배치 방식 | 판 크기 | 문화권 | 기원 |
|------|-----------|---------|--------|------|
| 바둑(圍棋) | 교차점 | 19×19 (361점) | 동아시아 | 중국, 기원전 ~2500년경 |
| 장기(將棋) | 교차점 | 9×10 (90점) | 한국 | 차투랑가 → 샹치 계열 |
| 샹치(象棋) | 교차점 | 9×10 (90점) | 중국 | 차투랑가 → 동양 전파 |
| 오목(五目) | 교차점 | 15×15 | 동아시아 | 바둑판 파생 |
| 체스 | 칸 | 8×8 (64칸) | 서양 | 차투랑가 → 샤트란지 → 체스 |
| 막룩 | 칸 | 8×8 | 태국 | 차투랑가 계열 |
| 센테레지 | 칸 | 8×8 | 에티오피아 | 차투랑가 계열 |
| 오셀로/리버시 | 칸 | 8×8 | 서양(영국/일본) | 근대 발명 |
| 쇼기(将棋) | **칸** | 9×9 (81칸) | 일본 | 차투랑가 계열 |

> **주목**: 쇼기는 동아시아임에도 '칸' 배치를 사용한다. 일본이 동아시아 교차점 전통에서 이탈한 예외인지, 혹은 별도의 전파 경로가 있는지 확인 필요.

### 1.2 차투랑가 분화론의 한계

차투랑가(6세기 인도)가 하나의 뿌리에서 동서양으로 갈라졌다는 설명은 널리 받아들여지지만, **교차점 vs. 칸의 분기**를 설명하기엔 불충분하다:

- 차투랑가 자체는 아쉬타파다(Ashtāpada, 8×8 칸) 보드 위에서 플레이됨 → **원형은 칸 배치**
- 즉, 동양 전파 과정에서 '칸 → 교차점'으로의 전환이 일어난 것
- 이 전환의 가장 유력한 원인: **바둑(圍棋)의 선행적 존재**

### 1.3 바둑의 영향 가설

바둑은 차투랑가보다 훨씬 오래된 게임(기원전 2500년경 추정)으로, 동아시아 보드게임 문화의 기저를 형성했다:

- 바둑은 처음부터 교차점 배치
- 샹치가 중국에 도입될 때, 이미 바둑판이라는 강력한 문화적 선례가 존재
- 바둑판의 교차점 체계가 샹치의 보드 디자인에 영향을 미쳤을 가능성이 높음
- 장기(한국)도 샹치의 영향 하에 교차점 배치를 계승

> **핵심 해석**: 교차점 vs. 칸의 분기는 "차투랑가의 동서 전파"가 아니라, **"바둑이라는 선행 그리드 문화가 동아시아에 존재했는가"**의 문제다.

## 2. 교차점과 칸: 왜 이것이 중요한가

### 2.1 공간 해석의 근본적 차이

이 차이는 단순한 규칙의 차이가 아니라, **공간을 어떻게 해석하느냐**의 차이:

- **칸(cell) 배치**: 공간은 분할된 영역(area)이다. 오브젝트는 영역을 '점유'한다.
  - 공간 = 면(面)의 집합
  - 존재 = 영역 점유

- **교차점(intersection) 배치**: 공간은 관계의 그물망이다. 오브젝트는 관계의 '결절점(node)'에 위치한다.
  - 공간 = 선(線)의 교차
  - 존재 = 관계 위의 위치

### 2.2 디자인적 시사점

그래픽 디자인의 그리드 시스템도 이 두 해석 사이에 있다:

- **서양 그리드 시스템** (뮐러-브로크만 등): 컬럼과 로우로 면을 분할, 요소는 칸 안에 배치
- **동양적 구성**: 여백(間, ma)과 관계성 중심 — 요소 사이의 '사이'가 구조를 만듦

> 이 관찰은 아직 직감 수준. 구체적 사례와 문헌 뒷받침 필요.

## 3. 그리드의 주술성

### 3.1 주술적 그리드의 사례들

그리드가 단순한 좌표계/레이아웃 도구를 넘어, 의례적·주술적 기능을 수행한 사례:

- **마방진(魔方陣, Magic Square)**: 숫자를 그리드에 배열하여 모든 행·열·대각선의 합이 동일하게 만드는 수학적 구조. 동서양 모두에서 부적·주술적 도구로 사용됨.
  - 중국 낙서(洛書): 3×3 마방진, 우주 질서의 상징
  - 이슬람 문화: 부적(talisman)으로서의 마방진
  - 뒤러의 <멜랑콜리아 I>(1514): 4×4 마방진 포함

- **만다라(Mandala)**: 동심원과 그리드의 결합. 명상·의례의 도구이자 우주론적 지도.
  - 티베트 모래 만다라: 그리드 기반 구성 → 완성 후 파괴
  - 힌두 바스투샤스트라(Vāstu Śāstra): 건축의 그리드가 곧 우주적 질서

- **풍수(風水)의 나경(羅經)**: 동심원 + 방사형 그리드. 공간의 기(氣)를 읽는 도구.

- **부적(符籍)**: 한국·중국·일본의 부적에서 그리드 형태의 문자/기호 배열. 글자의 위치 자체가 주술적 효력을 가짐.

- **카발라 세피로트(Sephirot)**: 10개의 결절점과 22개의 경로로 이루어진 트리 구조 — 그리드와 네트워크의 중간 형태.

### 3.2 세 가지 접근 방향 (메모에서)

1. **그리드 재조합 → 주술적 그리드**: 기존 그리드를 해체·재구성하여 주술적 의미를 부여
2. **주술적 그리드 → 병렬 구조**: 주술적 그리드들을 나란히 놓고 구조적 유사성 비교
3. **놀이 속 그리드를 주술적 그리드로 치환**: 체스/바둑 등의 그리드를 마방진/만다라 등으로 대체했을 때 무슨 일이 일어나는가

> ⚠️ 고전게임 그리드와 주술적 그리드가 계속 섞이면서 방향성이 모호해지는 문제. 하나를 선택하거나, 이 혼합 자체를 주제로 삼을 필요가 있음.

### 3.3 "그리드가 주술적이다"는 것의 의미

아직 정의가 필요한 개념. 잠정적 해석:

- 그리드에 요소를 배치하는 행위 자체가 **세계를 구성(구축)하는 행위**와 닮아 있다
- 마방진이 숫자의 '올바른 배치'로 조화를 만들 듯, 디자인 그리드도 요소의 '올바른 배치'로 질서를 만든다
- 차이: 주술은 배치가 **현실에 영향을 미친다**고 믿는 것. 디자인은 배치가 **인지에 영향을 미친다**는 것.
- 이 둘의 거리는 생각보다 가까울 수 있다

## 4. sound ↔ graphic 프로젝트와의 연결 가능성

[Sound ↔ Video ↔ Graphic 프로젝트](../2026-03-18_sound-video-graphic/findings.md)에서 다룬 UPIC 시스템, IanniX, 그리고 Wilcock의 playhead 방식은 모두 **그리드 위에서의 번역 규칙**이다:

- UPIC: X축=시간, Y축=피치 → 2D 그리드 위의 드로잉이 소리가 됨
- IanniX: 좌표계 위의 커서 이동이 MIDI/OSC 신호를 생성
- Wilcock의 playhead: 수직선이라는 최소 그리드가 시각→청각 번역의 규칙

만약 이 그리드를:
- 교차점 기반으로 재설계하면? (바둑판 위의 소리)
- 주술적 그리드(마방진 등)로 치환하면? (마방진의 숫자 배열이 음높이를 결정)

> 이 연결은 아직 직감 수준이지만, 그리드가 두 프로젝트를 잇는 **공통 구조**가 될 가능성이 있음.
