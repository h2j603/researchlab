# Sound ↔ Video ↔ Graphic — 비평

## 핵심 주장

**현재 상태: 모호함.**

이 리서치는 "소리와 그래픽 사이의 번역 규칙을 설계할 수 있다"는 전제에서 출발했지만, 현재는 여러 방향이 공존하며 핵심이 흐려져 있다:

- (a) 매체 간 번역의 역사를 조사한다 ← 완료됨, 충분한 양
- (b) 새로운 시각 언어를 설계한다 ← 아직 시작 안 됨
- (c) AI 파이프라인을 구축한다 ← 환경 제약으로 보류
- (d) 그리드가 번역의 열쇠다 ← 가설 단계, 검증 안 됨

**한 문장으로 써본다면**: "소리-시각 번역에서 그리드의 유형(교차점/칸/원형)이 결과를 결정한다."

→ 그런데 이걸 뒷받침하는 **실증이 하나도 없다.** 사례를 나열한 것과 주장을 증명한 것은 다르다.

## 범위 경계

**다루는 것**:
- 소리↔그래픽 번역의 역사적 사례와 방법론
- 번역 규칙(매핑)을 설계하기 위한 구조적 틀
- 그리드가 그 구조적 틀이 될 수 있는지의 가능성

**다루지 않아야 하는 것** (현재 침범 중인 영역):
- 동서양 보드게임 비교 문화론 ← 이건 그리드 프로젝트의 영역
- AI 도구 상세 스펙/파이프라인 설계 ← 현재 환경에서 실행 불가, 실행 가능해질 때 분리
- 공감각(synesthesia) 신경과학 ← 이미 충분히 다뤘고, 더 파면 이 프로젝트의 범위를 넘음

## 가장 약한 고리

1. **"그리드 유형이 소리 결과를 결정한다"는 주장에 실증이 없음.** Sara Adkins의 바둑판 시퀀서와 스펙트로그램이 다른 소리를 만든다는 것은 자명하지만, 그것이 "교차점 vs. 칸"이라는 프레임 때문인지, 단순히 다른 시스템이기 때문인지 구분되지 않음.

2. **금기서화 연결은 과잉 해석 가능성이 높음.** "네 분야가 한 방에 있었다"는 사실에서 "분리되지 않은 하나의 실천이었다"로의 도약에 근거가 부족함. 문인이 이 네 가지를 교양으로 갖췄다는 것과, 네 가지가 구조적으로 연결되어 있었다는 것은 완전히 다른 주장.

3. **실험 제안 A~D가 전부 사고실험.** 아이폰 환경에서 테스트할 수 있는 것이 없음. "바둑 기보를 스코어로 읽기"는 종이와 펜으로 가능하지만, 그 결과가 음악적으로 의미 있는지는 실제로 소리를 만들어봐야 알 수 있음.

## 반론

**"그리드가 핵심이다"가 틀렸다면:**

- 그리드는 매핑 규칙의 **시각화 도구**일 뿐, 매핑 규칙 **자체**가 아닐 수 있다. Wilcock의 Playhead가 강력한 건 그리드(수직선) 때문이 아니라, "차량 통과 = 소리 트리거"라는 **매핑 결정** 때문. 그리드를 바둑판으로 바꿔도 매핑 결정이 나쁘면 결과가 나쁨.
- 즉, 핵심 변수는 그리드의 형태가 아니라 **무엇을 무엇에 대응시키느냐**의 결정일 수 있음. 그리드에 집중하면 정작 매핑 규칙 설계를 소홀히 할 위험.

**"동양/서양 그리드 차이가 중요하다"가 틀렸다면:**

- 교차점 배치와 칸 배치의 차이가 보드게임에서는 의미 있을 수 있지만, 소리-시각 번역에서는 **기술적 구현의 차이**에 불과할 수 있음. "바둑판 위의 음악"이나 "체스판 위의 음악"이나, 좌표를 음높이로 변환하는 방식은 본질적으로 동일.

## 실행 가능성

현재 아이폰 환경에서 **실제로 할 수 있는 것**:

| 실험 | 가능 여부 | 비고 |
|------|----------|------|
| 관찰 → 리듬 기보 (손으로) | **가능** | 6-1절에서 제안한 "나만의 Playhead" 규칙 설계 |
| 바둑 기보를 종이 위에서 스코어로 읽기 | **가능** | 소리 없이 시각적 패턴만 먼저 분석 |
| 매핑 규칙 초안 작성 (메모) | **가능** | 시각 속성↔청각 속성 대응표 |
| Melobytes로 이미지→소리 변환 | **가능** | 브라우저 기반, 아이폰 Safari |
| 마방진 스펙트로그램 실험 | **불가** | DAW 또는 코딩 필요 |
| IanniX 체험 | **불가** | PC 필요 |
| 만다라 Playhead 구현 | **불가** | 코딩 필요 |

→ 가능한 것 중 **"바둑 기보를 종이 위에서 스코어로 읽기"**가 가장 구체적이고 이 리서치의 그리드 가설을 직접 테스트할 수 있는 실험.
