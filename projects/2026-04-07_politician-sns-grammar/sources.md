# 출처 정리

## 학술 논문 / 연구 보고서

1. **문재인 대통령의 SNS 정치 담화에 대한 텍스트언어학적 분석**
   - [KCI 논문](https://www.kci.go.kr/kciportal/landing/article.kci?arti_id=ART002666317)
   - 대통령 SNS의 텍스트언어학적 분석. 담화 구조, 언어 전략 분석.

2. **정치인 SNS의 메시지와 정치인 유형이 수용자의 반응에 미치는 영향**
   - [KCI 논문](https://www.kci.go.kr/kciportal/landing/article.kci?arti_id=ART002518274)
   - 정책/일상/공격 메시지 유형별 수용자 반응 차이 분석.

3. **정치인들의 선거 캠페인 수단으로서의 SNS 활용: 2015년 4·29 재·보궐선거를 중심으로**
   - [한국과학기술정보연구원](https://www.koreascience.kr/article/JAKO201615952962273.pub?lang=ko)
   - 후보자 SNS 메시지 내용 분석. '단순홍보'와 '주장' 전략이 지배적.

4. **국내 인터넷신문의 유명인 SNS 활용 기사의 현황과 문제점**
   - [KCI 논문](https://kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART002833225)
   - 정치인 SNS 기사에서 페이스북(70.4%) 압도적. 견해 표명, 폭로/비방 내용 중심.

5. **SNS의 확산과 선거·정당 정치지형의 변화** — 국회입법조사처
   - [PDF 보고서](https://www.nars.go.kr/fileDownload2.do?fileName=SNS+%ED%99%95%EC%82%B0%EA%B3%BC+%EC%84%A0%EA%B1%B0,+%EC%A0%95%EB%8B%B9%EC%A0%95%EC%B9%98+%EC%A7%80%ED%98%95%EC%9D%98+%EB%B3%80%ED%99%94_%EC%9E%A5%EC%9A%B0%EC%98%81.pdf&doc_id=97309)

6. **SNS 활용이 당선경쟁력에 미치는 영향: SNS의 미디어 효과 논의를 중심으로**
   - [KCI 논문](https://www.kci.go.kr/kciportal/landing/article.kci?arti_id=ART001736590)

7. **소셜 미디어와 정치 참여에 관한 연구: 사회적 압력이 정치 참여에 미치는 영향 분석**
