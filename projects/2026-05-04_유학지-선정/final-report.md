# 유학지 선정 — 최종 보고서

> **목적**: 학사 졸업 후(2028-09) 진학할 디자인 석사 프로그램 결정의 1차 정리.
> **상태**: 후보 4개 확정. 본인 추가 검증·공식 사이트 재확인 후 지원 단계 진입 예정.

---

## 0. 한 페이지 요약

**확정 후보 6개** (Bologna MA/MFA, 영어 트랙, 학사 졸업자에게 효율적):
1. **KABK Type and Media** (헤이그) — 활자 디자인 글로벌 톱, 1년, €11,488
2. **KABK Non Linear Narrative** (헤이그) — 리서치 기반 내러티브, 2년, €11,488/년
3. **Sandberg Instituut Design Department** (암스테르담) — 비평적 디자인 본진, 2년, ~€15,500/년
4. **Werkplaats Typografie** (Arnhem) — 인디 출판/체계 중심, 2년, €13,030/년 (50% 면제 가능)
5. **Design Academy Eindhoven — Critical Inquiry Lab** (Eindhoven) — 리서치 기반 비평적 디자인, 2년, €19,762/년
6. **Bauhaus-Universität Weimar MFA Media Art and Design** (Weimar) — 매체 실험, 2년, **무상**

**핵심 결정 논리**
- 비용 양극 구조: Bauhaus + Type and Media = 재정 안전 (각 ~3,800만원), KABK NLN + Werkplaats = 중간, Sandberg + DAE = 도전 (각 8,900~10,100만원)
- 학력 깔끔: 6개 모두 Bologna 시스템 → 홍익대 BA 위에 깔끔하게 누적
- 콘텐츠 다각화: 활자 / 내러티브 / 비평 / 출판 / 학문적 리서치 / 매체 — 본인 작업 방향의 6개 다른 각도
- KABK 두 프로그램 동시 지원 — 같은 학교 인프라 + 활자(모토)와 내러티브(콘텐츠) 양면 도전

**핵심 리스크**
- DAE/Sandberg/Werkplaats/KABK NLN는 한국 풀펀딩 장학금 없으면 진학 불가 또는 큰 자비 부담
- DAE는 가장 비쌈 (€19,762/년)
- Werkplaats 합격률 극히 낮음 (연 8명), KABK NLN "2년 실무 경험 선호" 약점
- KABK NLN 영상 비중 + 카메라 제약 충돌 가능성
- 6개 학교 모두 떨어질 시나리오 2~5%

## 2. 의사결정 여정

이 결정은 1차 라운드부터 5번의 변동을 거쳐 확정됐다. 각 변동은 새 정보·자기 진단의 결과:

1. **1차**: 국가 비교 (독일 / 네덜란드 / 스위스) → 스위스 탈락 (콘텐츠+생활비)
2. **2차**: 9개 학교 프로필 작성 → UdK Berlin 숨은 1순위 후보 발견
3. **3차**: 사용자 일정 + Sandberg 1순위 선언
4. **4차**: Sandberg 매칭 근거 자기 비판 → "왜 독일이 안 맞을까" 통찰
5. **5차**: 윤휘연(HfG Karlsruhe 졸업생) 발견 → 작업 기반 신호
6. **6차**: HfG Karlsruhe Diplom 검증 → 홍익대 BA 보유자에게 구조적 비효율
7. **7차 (최종)**: 4개 후보 확정 — 콘텐츠 우선, 트레이드오프 의식적 수용

이전 흔들림과 다른 점은: **이번 결정은 "키워드에 끌림"이 아니라 "트레이드오프를 의식적으로 수용"**.

## 5. 일정 (분기별)

### Phase 1 — 군복무 중 (2026-05 ~ 2027-02, 9개월)

**환경**: 아이폰만 가능. 텍스트·언어·사고 작업.

- 분기별: 5개 학교 인스타·졸업전시 모니터링, 한국 장학금 일정 추적

- **2027-07**: **암스테르담·헤이그·Arnhem 1차 방문** (졸업전시 시즌) — 콘텐츠 검증

## 9. 비평 — 이 결정의 약한 고리

### 9.1 콘텐츠 적합도 검증 부재
- "사용자 작업 = Sandberg/Werkplaats 결" 매칭은 **표면적 키워드**일 가능성. Phase 2 암스테르담·헤이그 방문에서 직접 졸업전시 보고 검증 필수.

### 9.2 Type and Media의 콘텐츠 미스매치
- 모토는 매칭하지만 사회 현상 리서치는 약함. 활자 디자이너로의 정체성 변경을 진심으로 받아들이는지는 별개 질문.

### 9.3 합격률 정량화 부재
- Werkplaats 8명, Sandberg 트랙별 변동, KABK 작은 정원. 정확 합격률 모름.

### 9.4 장학금 의존도 위험
- Sandberg/Werkplaats 갈 가능성은 한국 풀펀딩 수혜에 거의 전적으로 의존. 한국 장학금 합격률 미상.

### 9.5 영상/카메라 제약과 학교 결의 충돌
- Bauhaus Media Art와 KABK NLN은 영상 비중 큼. 군복무 중 영상 작업 못 함 → 전역 후 1년 안에 영상 포트폴리오 만들어야 함. 두 학교 모두 지원하려면 영상 작업 절대 필요.

### 9.6 KABK 두 프로그램 동시 지원의 함의
- 같은 학교에 두 프로그램 지원은 인프라 효율적이지만, 두 프로그램이 너무 다른 결을 평가하기 때문에 **포트폴리오 두 버전을 정말 잘 빌드해야** 효과 있음. 어설픈 양다리는 양쪽 다 떨어질 가능성.
- 또 인터뷰에서 "왜 두 프로그램 다 지원했냐"는 질문에 **본인 작업의 방향성을 두 가지 다른 언어로 설명**할 수 있어야 함.

### 9.7 메타 — 이 리서치가 본 리서치 랩 목적과 어울리는가
- 본 리서치 랩의 목적은 "사회 현상의 분류·분석을 디자인 작업의 개념적 토대로". "유학지 선정"은 진로 결정 메모. 이 프로젝트가 비대해지지 않게 관리 필요.
