## 비평 (Red Team)

### 이 표의 위험
- "10명 추출"이 진짜 목표가 아니라 **"학교의 리서치 방법론 패턴 파악"**이 진짜 목표였어야 함
- 졸업작 주제 표를 보고 본인 작업이 "여기 끼는가" 판단하는 건 표면적 매칭. 진짜 검증은 본인이 졸업전시 영상·작가 인터뷰를 직접 보면서 하는 것
- 학교별 작품 수 불균형 → "KABK NLN이 정합도 높음"이 진짜 그런 건지, 데이터 양 차이 때문인지 분리 안 됨

### 그럼에도 가치 있는 것
- KABK NLN 12명 표는 본인이 동기서 쓸 때 "내가 관심 있는 작업 방식이 NLN 학파 안에 자리 잡고 있다"는 구체적 증거가 됨
- 사용자 작업 방향("특정 대상 → 사회 구조 분석")이 NLN 학파의 표준 패턴과 **거의 그대로** 겹친다는 발견은 1차 라운드에서 못 본 것

### 학교 결정에 미칠 영향
이 데이터 기준이라면 **KABK NLN의 위치를 도전 트랙에서 더 강하게 둘 만한 근거**가 생긴다. 단, Sandberg·DAE 데이터가 없어서 비교 불완전. 사용자 PC 작업 후 다시 이 표 갱신 시점에 우선순위 재조정 가능.

# 부록 — 학교별 리서치 방법론 패턴

> 작품 주제만 보면 학교 결을 표면적으로 매칭하기 쉽다. 진짜 차이는 **"어떻게 리서치하는가, 어떤 도구로, 어떤 결과물로 변환하는가"**의 방법론에 있다. 이게 본인 작업과의 진짜 매칭을 가른다.

## 학교별 방법론 카드

### ① KABK Type and Media — "활자가 곧 리서치 도구"

**리서치 사이클**
1. 특정 활자/문자 형식 (역사적, 비라틴, 손글씨, 옛 활자) 선택
2. 손으로 그리거나 캘리그래피로 분석
3. 디지털 폰트 엔지니어링 (RoboFont, OpenType features)
4. 폰트 패밀리로 결과화

**특징적 도구**
- LettError 학파의 **RoboFont, DrawBot** (코드 기반 활자 디자인)
- OpenType variable fonts
- Python 스크립팅

**리서치 대상 패턴**
- 활자 자체가 사회·문화의 인덱스 (예: Nymfe System = 패션 + 활자)
- 한국인 졸업생 다수가 한국어/한자/한국 손글씨를 디지털화

**사용자와의 매핑**
- ⚠️ **방법론이 매우 specific하고 좁다**. "사회 현상 리서치"보다는 "활자 형식 리서치"
- ⭕ "최신 기술로 아날로그함" 모토와 직접 매칭 (손글씨 → 디지털)
- ⚠️ 본인이 활자 디자이너로 1년간 정체성 변경할 의향이 있어야 함

### ② KABK Non Linear Narrative — "개인 관찰을 다매체 내러티브로"

**리서치 사이클**
1. **개인 관찰·가족 기억·일상 사물·도시 디테일** 채택 (출발점이 자전적·미시적)
2. 그 사물/관계를 **거시 사회 구조로 확장** (식민, 자본, 전쟁, 가부장제 등)
3. 다매체 리서치 (다큐 인터뷰, 아카이브, 퍼포먼스, 책)
4. **영상 + 설치 + 출판** 3종 조합으로 결과화

**특징적 도구**
- 다큐멘터리 영상 (1~2채널 비디오 설치)
- 책/zine 형태의 "리서치 노트"
- 퍼포먼스·집회 (2024~2025 부상)

**리서치 대상 패턴**
- **특정 사물/장소/관계** (은수저, 감옥 건물, "Palace" 상점, 정원, 가족) → 사회 구조의 매개
- 언어·이름짓기의 정치 (오키나와어, 헤이그 도시 어휘, 신체 어휘집)
- 신체·집회·체현 (참여형 리서치)

**사용자와의 매핑**
- ⭕ **"고유한 분류 규칙으로 추상 주제를 작업물로 전환" — 사용자 진술과 가장 정확히 매칭**
- ⭕ 사용자 기존 작업(`evangelion-org-design`, `serious-nonsense-sos-fgl`)이 정확히 이 학파 결
- ⚠️ 영상/다큐 비중 큼 (사용자 카메라 제약과 충돌 가능, 전역 후 보완 필요)

### ③ Sandberg Instituut — "정치 의제를 폼으로 들이밀기"

**리서치 사이클**
1. **시대 정치 의제** 선택 (식민주의, 환경 위기, 자본주의, 젠더, AI)

4. 출판물·이벤트·설치로 폼화

**특징적 도구**
- 매니페스토·선언문·소책자
- 큐레이션 (학생이 자기 전시·이벤트 운영)
- 객원 강사 다수 (튜터 풀이 매년 바뀜)

**리서치 대상 패턴**

**사용자와의 매핑**

- ⚠️ **"고유한 분류 규칙·매뉴얼"보다 "비평적 입장 표명"** 결이라 사용자 시스템적 사고와 미스매치 가능
- ⚠️ Temporary Programme이 본인 방향과 안 맞으면 합격해도 의미 약해짐

### ④ Werkplaats Typografie — "출판물이 곧 리서치"

**리서치 사이클**
1. 외부 클라이언트 **commission**으로 시작 (학생이 받음)
2. 현장 조사·인류학적 관찰 (특정 동네·커뮤니티·사물)
3. 책/zine/잡지 형식 안에서 **편집·구조·형식 자체를 리서치**
4. **인쇄물 자체가 결과물 + 리서치 자료** (분리 안 됨)

**특징적 도구**
- 책 디자인, 인쇄 실험, 종이 선택, 제본
- 편집 (텍스트와 이미지의 조율)
- 활자, 그리드

**리서치 대상 패턴**
- 도시 공간·이민자 커뮤니티·일상 물건·형식 그 자체
- "왜 이 형식인가"가 핵심 — 내용과 형식이 분리 안 됨
- Roma Publications 학파의 미적 코드 강함

**사용자와의 매핑**
- ⭕ 사용자 `grid-systems`, `opensource-font-github` 같은 체계·시스템 작업과 결 매칭
- ⭕ 영상 부담 없음 (출판 중심)
- ⚠️ "외부 commission 기반"이 사용자의 자기 주도형 리서치 방식과 충돌 가능
- ⚠️ 미적 관습 강해서 본인 색 잃기 쉬움 ("Roma 스타일" 흡수 위험)

## 사용자 작업 방향 × 6개 학교 방법론 매핑

사용자의 자기 진술:
- "사회 현상을 자기만의 방식으로 리서치"
- "고유한 논리, 규칙, 매뉴얼로 분류·분석"
- "리서치 결과를 디자인 작업의 개념적 토대로"
- "최신 기술로 최고의 아날로그함"

| 진술 요소 | 가장 매칭되는 학교 | 매칭 근거 |
|---|---|---|
| "사회 현상을 자기 방식" | **KABK NLN** | 개인 출발점 + 자율성 + 다매체 결과 |
| "고유한 논리·규칙·매뉴얼로 분류·분석" | **DAE CIL** = NLN | DAE 학문적·체계적 / NLN 자전적·체계적 |
| "리서치 결과 → 개념 토대" | DAE CIL = NLN = Sandberg | 셋 다 리서치-디자인 직결 |
| "최신 기술 + 아날로그함" | **KABK T&M** = Bauhaus Weimar | T&M 활자 / Bauhaus 매체 실험 |
| 종합 1순위 | **KABK NLN** | 4개 진술 중 3개 만족 |
| 종합 2순위 | **DAE CIL** | 학문적 결이 추가될 경우 |

### 발견 — 사용자 작업이 두 학파에 걸쳐 있다

너의 진술은 **"NLN 방식 (자전적·미시적·다매체) + DAE 방식 (학문적·체계적·시각화)"의 교차점**에 있다. 이게 둘을 동시 지원하는 진짜 이유.

만약 둘 다 합격하면:
- **NLN을 택하면** → 본인의 자전적·구체적 관찰을 깊이 발전
- **DAE를 택하면** → 본인의 체계적·분석적 사고를 학문적으로 발전

지금은 본인이 어느 쪽으로 더 끌리는지 모르는 게 정상. **Phase 2~3에서 졸업전시를 직접 보고 결정하면 된다.**

4. **"외부 의제(시대 정치)에 정렬"이 좋은가, "자기 의제 자유롭게 추구"가 좋은가?**

이 4개 질문에 대한 본인 답이 6개 학교 우선순위를 결정한다.

## 한 줄 정리 — 방법론 기준 우선순위

데이터 기준으로 사용자에게 가장 정합도 높은 학교 순서:

1. **KABK NLN** — 자전적·체계적, 자율성 높음
2. **DAE CIL** — 학문적·체계적, 학비·학파 흡수 우려
3. **Werkplaats** — 출판 중심, 영상 부담 없음, 학파 강함
4. **Sandberg** — 정치 결, Temporary Programme 정렬 필요
5. **KABK T&M** — 모토 매칭이지만 사회 리서치는 약함
6. **Bauhaus Weimar** — 매체 실험, 그래픽 디자인 거리 있음

단, **Sandberg·DAE·Bauhaus는 졸업작품 데이터 부족**으로 평가 불완전. 사용자 PC 작업으로 보완 필요.
