# 유학지 선정 — 독일 vs 네덜란드

## 한 줄 요약
**비용 1순위는 독일, 콘텐츠 적합도 1순위는 네덜란드.** 둘 다 살리려면 독일을 메인 트랙으로 두고 네덜란드는 장학금 도전 트랙으로 병행한다.

## 전제 조건 (사용자 확정)
- 학비 부담은 불가 → 비EU 국제학생 풀 학비 못 냄
- 후보국: 독일 / 네덜란드 / 스위스 → **스위스 탈락** (생활비 폭탄, 콘텐츠 미스매치)
- 독일어 학습 의향: 있음 (단, 부담은 큼)
- 장학금 도전 의지: 있음
- 학사 전공: 디자인

## 결정 트리
```
1. 메인 트랙 (안전망)
   → 독일 영어 프로그램 학교 + DAAD 장학금 도전
2. 도전 트랙 (콘텐츠 최적)
   → Sandberg / Werkplaats / KABK + 학교 자체 fellowship + 한국 측 장학금
3. 둘 다 지원하고 결과 보고 결정 (병행 권장)
```
