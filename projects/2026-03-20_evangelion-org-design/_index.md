# 에반게리온의 NERV와 SEELE 조직 설계 분석

## 리서치 목적
"허구의 조직이 어떻게 실재하는 것처럼 느껴지는가"를 에반게리온의 NERV/SEELE를 통해 분석한다.
궁극적으로 **허구적 권위의 디자인 기법**을 추출하는 것이 목표.

## 문서 구조
- [findings.md](findings.md) — 상세 조사 내용 (5개 섹션 + 디자인 시사점 표)
- [_critique.md](_critique.md) — 비평 (Red Team)

## 핵심 질문
1. NERV의 조직 아이덴티티(로고, 슬로건, 조직도)는 어떻게 설계되었는가? -> **조사 완료**
2. SEELE는 "보이지 않는 권위"를 어떻게 연출하는가? -> **조사 완료**
3. 카발라/사해문서/유대 신화가 세계관에 어떻게 작동하는가? -> **조사 완료**
4. "헛소리를 진지하게" 만드는 구체적 기법은 무엇인가? -> **조사 완료**
5. 안노 히데아키는 이 설정을 왜, 어떻게 만들었는가? -> **조사 완료**

## 추출된 기법 요약
| 기법 | 핵심 원리 |
|------|-----------|
| 이국어 명명 | 익숙하지 않은 언어가 자동적 권위를 생성 |
| 불완전 정보 | 빈 공간이 수용자의 해석을 유발 |
| 실존 차용 | 실제 문헌/명칭 + 허구 내용 = 진위 혼동 |
| 관료적 형식 | 형식이 갖춰지면 내용이 실재처럼 보임 |
| 상징 과잉 | 상징이 많으면 "의미가 있을 것"이라는 착시 |
| 모순적 슬로건 | 표면/이면의 괴리가 깊이의 착시를 만듦 |
