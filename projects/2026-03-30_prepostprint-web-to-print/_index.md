# PrePostPrint — 웹 기술로 만드는 인쇄 디자인

## 프로젝트 개요
HTML, CSS, JavaScript 등 웹 프로그래밍 언어를 디자인 및 셀프 퍼블리싱 도구로 활용하는 커뮤니티/운동인 PrePostPrint에 대한 심층 리서치. 웹 기술과 인쇄 디자인의 교차점을 탐구한다.

## 핵심 발견 요약
- PrePostPrint은 **프랑스/벨기에 중심**의 비공식 커뮤니티로, 웹 기술로 인쇄물을 만드는 대안적 출판 운동
- 2017년 파리 Gaîté Lyrique에서 에디션 살롱 개최, 2018년 Paged.js와의 합동 워크숍이 핵심 전환점
- **Julie Blanc, Fred Chasen, Sarah Garcin, Raphaël Bastide** 등이 핵심 인물
- Paged.js는 이 커뮤니티와 직접적으로 연결된 도구, Vivliostyle는 같은 철학 공유하나 별도 프로젝트
- 2023-2024 메인테이너 교체가 있었으며, 최근(2024.9) PPPermapublishing 행사 이후 2025-2026 일정은 미확인
- **"최신 기술로 최고의 아날로그함"** 철학과 정확히 맞닿아 있는 운동
