# 마을 표지석 — 패싯 분류 (Faceted Classification)

단일 위계(예: "제주형 / 도시형")는 한 표지석이 가진 여러 성질을 한 줄로
뭉갠다. 대신 **서로 독립적인 축(패싯)** 으로 분해한다. 각 표지석은
"F1=화산암계, F4=도색·백, F6=지명+유래…" 처럼 좌표로 기술되고,
어느 축으로든 가로질러 묶을 수 있다.

## 패싯 스킴 (통제 어휘)

| 코드 | 패싯 | 값(통제 어휘) | 근거 신뢰도 |
|------|------|----------------|--------------|
| **F1** 석재군 | 돌의 암석 계열 | `현무암·스코리아` / `적갈 풍화화산암` / `회색 화강암` / `흑회색 화강암` / `풍화 화강암(베이지)` | 계열 구분 中, 세부 암석명 低 |
| **F2** 가공도 | 원석에 가한 인공 | `자연석(원형 유지)` / `부분가공` / `완전가공(정형)` | 高 |
| **F3** 비례·자세 | 덩어리의 자세 | `수직 첨탑` / `수직 비석` / `수직 괴석` / `등신대 괴석` / `수평 와상` / `수평 캔틸레버` / `장식 실루엣` | 高 |
| **F4** 글자 | 새김 기법·색·방향 | 기법 `음각`/`도색` × 색 `무채(흑)`/`백`/`금` × 방향 `세로`/`가로` | 高 |
| **F5** 기단 | 돌을 받치는 구조 | `없음(지면직립)` / `자연석·막돌` / `가공석 기단` / `일체형` / `별도 명판석` | 高 |
| **F6** 문구 정보층 | 새겨진 정보의 종류 | `지명만` / `지명+유래(연혁)` / `지명+사회관계` / `지명+개발실적` | 高 |
| **F7** 입지 | 놓인 장소의 성격 | `농어촌 마을입구` / `간선 도로변` / `해안 광장` / `아파트단지 입구` / `도시 가로·공원` | 高 |
| **F8** "마을" 지시대상 | '마을'이 가리키는 실체 | `현존 행정리` / `소멸 자연마을의 기억` / `분양·단지 단위` | 中(추론) |
| **F9** 지역 | 행정구역 | `제주 서부(한경·대정)` / `인천 연수` / `서울 강남` / `미특정` | 9개 高, 6개 미특정 |

> F8은 추론 패싯이다. 문구·입지에서 역산한 값이며 단정 아님.
> F1 세부 암석명과 F9 미특정 항목은 `_critique.md`의 약한 고리 참조.

## 패싯 교차 읽기 (관찰 — 단정 아님, n=15)

위계 분류였다면 안 보였을 가로축 묶음:

- **F4 글자 × F1 석재**: `도색`은 화산암계·흑회색 비석(1·2·3·13·14·15)에,
  `음각`은 회색 화강암(4~12)에 거의 갈린다. 색이 안 들어가는 검은 돌엔
  칠하고, 무른 회색 돌엔 판다 — 재료가 기법을 강제.
- **F6 문구 × F8 지시대상**: `개발실적`(9·11)은 예외 없이 `분양·단지`.
  표지석에 공사 주체를 새기는 순간 그건 마을이 아니라 부동산이다.
- **F2 가공도 × F7 입지**: `완전가공`(10·12)은 전부 도시. 자연석
  그대로(자연석)는 제주·도로변. 도시일수록 돌을 깎는다.
- **F8만 따로**: `현존 행정리`=제주에 몰림, `소멸 자연마을의 기억`=서울
  강남·인천, `분양·단지`=아파트. 같은 "마을"이 지역에 따라 시제가 다름
  (지속 / 추모 / 신축).

## 사용법
- 한 축으로 정렬: 예) F8=`소멸 자연마을` → 4·5·8·13·14·15 (도시 재개발이
  옛 지명을 박제한 표본군).
- 두 축 교차: 예) F4=`음각` ∧ F7=`아파트단지` → 9·10·11 (도시 개발형의
  글자 관습).
- 새 표본은 같은 9개 패싯 값만 채우면 자동으로 이 격자에 들어온다.
