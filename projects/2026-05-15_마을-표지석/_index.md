# 마을 표지석 프로젝트

마을 입구·아파트 단지·도시 가로에 세워진 이름돌(표지석)을 수집·분류하는
현장 기록 프로젝트. 돌의 재질·형태·크기와 지명·문구를 목록화해, 한국의
장소 표시 관습이 농촌 자연석에서 택지개발 기념물로 전용되는 양상을 본다.

## 현황
- 수집: **18개** (제주 4, 내륙 자연석 6, 도시/택지 8) + 경계 제외 1
- 위치 9개 확정 / 9개 미특정. 돌·크기는 **사진 기반 추정** — `confidence: low`
