# 마을 표지석 카탈로그 — 1차 수집 15개

> 사용자가 직접 수집한 표지석 사진(Are.na 보드, 비공개)을 기반으로
> **지명 / 추정 돌 종류 / 추정 크기**를 문서화한 1차 카탈로그.
> 모든 돌 종류·크기·위치는 **사진 관찰에 의한 추정치**이며 현장 실측이 아니다.
> 신뢰도 표기와 한계는 아래 [방법론·신뢰도](#방법론신뢰도) 참조.

## 배경 및 목적
"마을 표지석 프로젝트"의 자료 정리 단계. 마을 입구·아파트 단지·도시
가로에 세워진 이름돌(표지석)을 수집해, 돌의 재질·형태·크기와 지명·문구를
분류하여 작업의 개념적 토대로 삼는다. 이 문서는 해석이 아니라 **목록화**가
목적이다. 의미 분석은 `_critique.md`에서 범위를 통제한다.

### 9. 비류마을 (별도 명판석)
```
공 사 명 : 인천연수지구 제1단계 부지조성공사 (1공구)
공사구간 : 인천시 남구 연수동·청학동 일원
공사기간 : 1988.12.30 ~ 1993.10.31
시 행 청 : 한국토지개발공사
시공회사 : 두산건설(주)
```
→ 마을의 역사가 아니라 **택지개발 공사 실적**을 기록. 표지석이 마을 정체성이
아니라 개발 주체의 기념물로 기능하는 사례.

### 14. 자곡동 교수마을 (연혁문)
```
자곡동은 紫陽洞의 紫와 池谷里의 谷을 따서 자곡리로 불려오다
현재의 자곡동으로 불리었고, 택지조성이후 학식과 덕망이
높은 대학교수가 많이 살고 있어 교수마을이라고 불리었다.
```

### 15. 자곡동 쟁골마을 (연혁문, 일부 가림)
```
쟁골마을은 원래 자양골의 준말로, 자양골은 서산을 등지고
동쪽을 향한 마을로 날이 밝으면 제일 먼저 돋는 해의 붉은
볕을 받는 마을이라 불려진 것이며, 현재 [지]명인 자곡동은
紫陽洞의 紫와 池谷里의 谷을 따서 만든 [이름]이다.
```

## 1차 관찰 (가설 — 검증 전)

세 가지 유형이 구분된다. **단정하지 말 것.** 표본 15개, 사용자가 임의 수집한
편향 표본이다.

1. **제주 현무암형 (1·2·3)** — 마을 입구, 검거나 적갈색 화산암 자연석,
   도색 글씨. 돌이 곧 그 땅의 지질.
2. **내륙 화강암 자연석형 (4·5·6·7)** — 농촌·도로변, 회색 둥근 강돌,
   음각. 돌과 장소의 지질 연관은 약함(어디서든 화강암).
3. **도시 아파트·택지형 (8~15)** — 단지 입구·도시 가로, 가공도가 높고,
   문구가 마을 유래 또는 **건설사·공사 실적**. 표지석이 부동산 개발의
   기념물로 전용된 형태.

표기 불일치 관찰: 13번 상단은 "지곡동", 14·15번은 "자곡동". 연혁문은
한자 `池谷里`(지곡리)에서 유래를 설명하면서 정작 한글 표기는 "자곡"으로
적혀 있다. 같은 행정동 내 표지석들끼리 표기가 어긋나는 현상 — 추후 확인 대상.

## 방법론·신뢰도

- **출처**: 사용자가 직접 촬영·수집한 1차 자료(현장 사진). 외부 문헌 아님.
- **돌 종류**: 사진의 색·표면 질감으로 추정한 **육안 분류**. "화강암 계열 /
  현무암 계열"의 큰 구분은 비교적 신뢰 가능하나, 세부 암석명(응회암·조면암·
  섬록암 등)은 `speculative`. 확정하려면 현장 관찰 또는 지질도 대조 필요.
- **크기**: 가드레일(~0.8 m)·버스정류장·차량·사람 등 사진 내 기준물 대비
  **개략 추정**. 오차 ±30 % 이상 가능. 실측 아님.
- **위치**: 명판 문구로 확인된 9번(인천연수지구)·14·15번(자곡동) 외에는
  배경·돌 재질로 **추정**한 것. 1·2·3번의 제주 비정은 화산암+오름 배경 근거,
  6번은 미확정.
