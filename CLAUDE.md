# Research Lab — Claude 협업 가이드

## 이 리포지토리의 목적
이 리포지토리는 사용자의 개인 리서치 랩입니다.
Claude와 함께 다양한 주제를 깊이 있게 조사하고, 결과를 체계적으로 정리하는 공간입니다.

## 작업 원칙

### 리서치 진행 방식
- 새로운 리서치 주제는 `research/` 디렉토리 아래에 주제별 폴더를 만들어 진행한다
- 폴더명은 `YYYY-MM-DD_주제명` 형식을 따른다 (예: `2026-03-18_llm-scaling-laws`)
- 각 리서치 폴더에는 반드시 `README.md`(요약)와 `findings.md`(상세 내용)를 포함한다
- 완료된 리서치는 `archive/`로 이동한다

### 품질 기준
- 출처가 있는 정보는 반드시 출처를 명시한다
- 주장과 사실을 구분하여 기술한다
- 불확실한 정보는 명시적으로 "불확실" 또는 "추가 검증 필요"로 표시한다
- 한국어로 작성하되, 기술 용어는 영어 병기를 허용한다

### 결과물 형식
- 마크다운(.md)을 기본 형식으로 사용한다
- 코드 예제가 포함될 경우 적절한 언어 태그를 사용한다
- 다이어그램이 필요한 경우 Mermaid 문법을 사용한다

## 디렉토리 구조

```
researchlab/
├── CLAUDE.md          # 이 파일 — Claude 협업 규칙
├── README.md          # 리서치 랩 소개
├── research/          # 진행 중인 리서치
├── archive/           # 완료된 리서치
├── notes/             # 빠른 메모, 아이디어
└── templates/         # 리서치 템플릿
```

## 커밋 메시지 규칙
- `research: 주제명 - 내용` — 리서치 관련
- `notes: 내용` — 메모 관련
- `chore: 내용` — 설정/구조 변경
