# Research Lab

Claude와 함께 운영하는 개인 리서치 랩입니다.

## 사용법

1. 새 리서치 주제가 있으면 `research/` 아래에 날짜_주제명 폴더를 생성합니다
2. 조사 결과를 `findings.md`에, 요약을 `README.md`에 정리합니다
3. 완료된 리서치는 `archive/`로 이동합니다
4. 빠른 메모는 `notes/`에 저장합니다

## 구조

| 디렉토리 | 용도 |
|-----------|------|
| `research/` | 진행 중인 리서치 |
| `archive/` | 완료된 리서치 |
| `notes/` | 빠른 메모, 아이디어 |
| `templates/` | 리서치 템플릿 |
